# 現值利息因子

<ArticleMeta id={102429} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

現值利息因子 (PVIF) 是一種用於估計將在未來某個日期收到的一筆資金的當前價值的公式。PVIF 通常以表格形式呈現，其中包括不同時間段和利率組合的值。
