# 二元期權

<ArticleMeta id={100410} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

二元期權是一種金融產品，交易雙方根據期權是否在到期時 “有價值” 而被分配為兩種不同的結果。二元期權的結果取決於 “是” 或 “否” 的命題結果，因此得名 “二元”。如果二元期權到期時有價值，交易者將獲得支付，如果二元期權到期時沒有價值，交易者將遭受損失。
