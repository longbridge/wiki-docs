# 股東增值

<ArticleMeta id={100496} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

股東增值是指公司創造的超額經營利潤與資本成本或資本成本之間的差額。基本計算公式為税後淨經營利潤 (NOPAT) 減去資本成本，資本成本根據公司的加權平均資本成本來計算。
