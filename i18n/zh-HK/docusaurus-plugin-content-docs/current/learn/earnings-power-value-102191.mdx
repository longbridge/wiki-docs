# 盈利能力價值

<ArticleMeta id={102191} updatedAt={'2024-09-12 02:31:52'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>盈利能力價值（Earnings Power Value, EPV）是一種評估公司內在價值的財務指標，它通過預測公司未來的持續盈利能力來估算其當前價值。EPV 基於公司的當前收益，並假設這些收益在未來能夠持續不變。與傳統的市盈率（P/E）或淨現值（NPV）方法不同，EPV 更注重公司現有的盈利能力和業務穩定性，而不是未來的增長潛力。</p><p>盈利能力價值的計算公式為：<br/>EPV = 調整後的淨收益/資本化率</p><p>其中，調整後的淨收益是將公司當前的淨收益調整為長期可持續水平的結果，資本化率通常是公司要求的回報率或折現率。</p><p>主要特點包括：</p><p>關注現有盈利能力：EPV 側重於公司當前的盈利水平，而非未來增長預測。<br/>簡化假設：假設公司當前的盈利能力能夠持續，簡化了對未來不確定性的預測。<br/>保守估值：由於不考慮未來的增長，EPV 通常提供相對保守的估值。<br/>適用性廣：適用於評估成熟且盈利穩定的企業，尤其是那些沒有顯著增長潛力的企業。<br/>盈利能力價值的應用示例：<br/>假設某公司的調整後淨收益為 5000 萬美元，資本化率為 10%。該公司的 EPV 為：<br/>EPV = 5000 萬美元/0.10 = 5 億美元</p>
