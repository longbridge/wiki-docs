# 相同性質資產

<ArticleMeta id={102160} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

相同性質資產是指兩個房地產資產，無論等級或質量如何，都可以在不產生任何税務責任的情況下交換。國內税收法典將相同性質資產定義為在 1031 條款下為投資、貿易或業務目的而持有的任何資產，使它們成為 1031 交換。這意味着交換涉及的兩個物業都必須用於商業或投資目的，因此個人住宅不符合相同性質資產的資格。
