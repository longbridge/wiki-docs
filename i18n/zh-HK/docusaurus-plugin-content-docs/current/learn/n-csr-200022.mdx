# N-CSR

<p>N-CSR 基金年度報告：投資公司提交的年度報告，包含經審計的財務報表和投資組合的詳細信息</p>

<AIContent content={`<p><strong>定義：</strong><br/>N-CSR 基金年度報告是投資公司提交的年度報告，包含經審計的財務報表和投資組合的詳細信息。N-CSR 是 “Certified Shareholder Report” 的縮寫，主要用於向投資者披露基金的財務狀況和投資表現。</p><p><strong>起源：</strong><br/>N-CSR 報告的起源可以追溯到美國證券交易委員會（SEC）在 2004 年實施的規則，旨在提高投資公司信息披露的透明度和準確性。該規則要求投資公司每年提交一次 N-CSR 報告，以確保投資者能夠獲得最新的財務信息。</p><p><strong>類別與特點：</strong><br/>N-CSR 報告主要分為兩類：年度報告和半年度報告。年度報告包含完整的經審計財務報表，而半年度報告則包含未經審計的財務信息。<br/>特點包括：<ul><li>詳細的財務報表：包括資產負債表、收益表和現金流量表。</li><li>投資組合明細：列出基金持有的所有證券及其市值。</li><li>管理層討論與分析：提供對基金表現的深入分析和未來展望。</li></ul></p><p><strong>具體案例：</strong><br/>案例一：某投資公司提交的 N-CSR 報告顯示，其在過去一年中實現了 10% 的回報率，主要得益於其在科技股的投資。報告詳細列出了該公司持有的所有科技股及其市值，幫助投資者瞭解其投資策略。<br/>案例二：另一家基金公司在其 N-CSR 報告中披露了其在新興市場的投資組合，並解釋了這些市場的高增長潛力和風險。通過詳細的財務報表和投資組合明細，投資者可以清楚地看到基金的投資方向和風險管理策略。</p><p><strong>常見問題：</strong><br/>1. 為什麼 N-CSR 報告如此重要？<br/>答：N-CSR 報告提供了基金的詳細財務信息和投資組合，幫助投資者做出明智的投資決策。<br/>2. N-CSR 報告與普通的年度報告有何不同？<br/>答：N-CSR 報告是專門為投資公司設計的，包含更詳細的投資組合信息和經審計的財務報表。</p>`} id={200022} />
