# 穩定價值基金

<ArticleMeta id={102148} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

穩定價值基金是一種由債券組成的投資組合，投資者對利率下降或資本損失進行保險。穩定價值基金的所有者將繼續收到協議的利息支付，無論經濟狀況如何。穩定價值基金是一些退休計劃（如公司 401(k) 計劃）中的常見選擇，特別針對那些接近退休的儲蓄者。
