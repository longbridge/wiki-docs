# 或有價值權

<ArticleMeta id={100032} updatedAt={'2024-09-13 11:53:19'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>術語 “或有價值權（CVR）” 指的是一種權利，通常授予面臨重組或收購的公司股東。這些權利確保股東在特定事件發生時獲得一定的利益，通常在指定的時間框架內。</p><p>這些權利類似於期權，因為它們通常有一個到期日，超過此日期後，權利將不再適用。CVR 通常與公司股票的表現相關。</p>

<AIContent content={`<h2>或有價值權（CVR）</h2><p><strong>定義：</strong>術語 “或有價值權（Contingent Value Rights, CVR）” 指的是一種權利，通常授予面臨重組或收購的公司股東。這些權利確保股東在特定事件發生時獲得一定的利益，通常在指定的時間框架內。CVR 類似於期權，因為它們通常有一個到期日，超過此日期後，權利將不再適用。CVR 通常與公司股票的表現相關。</p><h3>起源：</h3><p>或有價值權的概念起源於 20 世紀 80 年代，當時公司重組和收購活動頻繁。為了保護股東利益，企業開始引入 CVR 作為一種補償機制，確保股東在特定條件下能夠獲得額外的收益。隨着時間的推移，CVR 逐漸成為併購交易中的常見工具。</p><h3>類別與特點：</h3><p>CVR 可以分為兩大類：現金支付型和股票支付型。<ul><li><strong>現金支付型：</strong>在特定事件發生時，股東將獲得現金支付。這種類型的 CVR 通常與公司業績指標或特定項目的成功掛鈎。</li><li><strong>股票支付型：</strong>在特定事件發生時，股東將獲得額外的公司股票。這種類型的 CVR 通常與公司股價表現相關。</li></ul>CVR 的主要特點包括：<ul><li><strong>條件性：</strong>CVR 的價值取決於特定事件的發生，如公司達到某一業績目標或完成某一項目。</li><li><strong>時間限制：</strong>CVR 通常有一個到期日，超過此日期後，權利將不再適用。</li><li><strong>風險與回報：</strong>CVR 為股東提供了額外的潛在收益，但也伴隨着一定的風險，因為特定事件可能不會發生。</li></ul></p><h3>具體案例：</h3><p><strong>案例一：</strong>在某次製藥公司併購中，收購方承諾如果被收購公司的一款新藥在未來兩年內獲得 FDA 批准，股東將獲得每股 2 美元的現金支付。這種 CVR 確保了股東在新藥成功上市時能夠獲得額外收益。<br/><strong>案例二：</strong>某科技公司在重組過程中，向股東發行了股票支付型 CVR。如果公司在未來三年內股價達到特定目標，股東將獲得額外的公司股票。這種 CVR 激勵股東支持公司的長期發展。</p><h3>常見問題：</h3><p><strong>1.
