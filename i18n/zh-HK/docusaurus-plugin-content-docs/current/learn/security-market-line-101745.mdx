# 證券市場線

<ArticleMeta id={101745} updatedAt={'2024-09-12 04:07:01'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

證券市場線（SML）是一條畫在圖表上的線，它是資本資產定價模型（CAPM）的圖形表示，顯示了各種可交易證券的系統風險或市場風險的不同水平，與任何給定時間點上整個市場的預期回報相對應。也被稱為 “特徵線”，SML 是 CAPM 的可視化，圖表的 x 軸代表風險（以 beta 為單位），y 軸代表預期回報。給定證券的市場風險溢價由其相對於 SML 在圖表上的位置決定。

<AIContent content={`<p><strong>定義：</strong>證券市場線（SML）是一條畫在圖表上的線，它是資本資產定價模型（CAPM）的圖形表示，顯示了各種可交易證券的系統風險或市場風險的不同水平，與任何給定時間點上整個市場的預期回報相對應。也被稱為 “特徵線”，SML 是 CAPM 的可視化，圖表的 x 軸代表風險（以 beta 為單位），y 軸代表預期回報。給定證券的市場風險溢價由其相對於 SML 在圖表上的位置決定。</p><p><strong>起源：</strong>證券市場線的概念源自資本資產定價模型（CAPM），該模型由威廉·夏普（William Sharpe）在 1960 年代提出。CAPM 的目的是通過考慮系統性風險來估算證券的預期回報，從而幫助投資者做出更明智的投資決策。</p><p><strong>類別與特點：</strong>證券市場線主要有以下幾個特點：<ul><li><strong>系統風險：</strong>SML 只考慮系統風險（市場風險），而不考慮非系統風險（個別風險）。</li><li><strong>線性關係：</strong>SML 展示了風險與預期回報之間的線性關係，風險越高，預期回報也越高。</li><li><strong>市場均衡：</strong>在市場均衡狀態下，所有證券的預期回報都應位於 SML 上。</li></ul></p><p><strong>具體案例：</strong><ol><li><strong>案例一：</strong>假設某投資者正在評估兩隻股票 A 和 B。股票 A 的 beta 值為 1.2，股票 B 的 beta 值為 0.8。假設無風險利率為 2%，市場的預期回報率為 8%。根據 SML 公式，股票 A 的預期回報率為 2% + 1.2 * (8% - 2%) = 9.2%，股票 B 的預期回報率為 2% + 0.8 * (8% - 2%) = 6.8%。</li><li><strong>案例二：</strong>某基金經理發現一隻股票 C 的預期回報率為 10%，其 beta 值為 1.5。根據 SML 公式，假設無風險利率為 2%，市場的預期回報率為 8%，則股票 C 的預期回報率應為 2% + 1.5 * (8% - 2%) = 11%。由於實際預期回報率低於 SML 上的回報率，基金經理可能會認為這隻股票被高估了。</li></ol></p><p><strong>常見問題：</strong><ul><li><strong>問：</strong>為什麼 SML 只考慮系統風險？<br/><strong>答：</strong>因為系統風險是無法通過分散投資來消除的，而非系統風險可以通過多樣化投資組合來降低。</li><li><strong>問：</strong>如果某隻證券的預期回報率低於 SML 上的回報率，意味着什麼？<br/><strong>答：</strong>這通常意味着該證券被高估了，投資者可能會選擇賣出該證券。</li></ul></p>`} id={101745} />
