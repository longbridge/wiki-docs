# 前端銷售費

<ArticleMeta id={101571} updatedAt={'2023-09-26 13:58:58'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

前端銷售費是在初始購買投資時收取的佣金或銷售費用。這個術語最常用於共同基金投資，但也可能適用於保險政策或年金。前端銷售費從初始存款或購買資金中扣除，從而降低實際進入投資產品的金額。前端銷售費支付給金融中介機構，作為找到並銷售與客户需求、目標和風險承受能力最匹配的投資的補償。因此，這些是一次性收費，不是投資的持續運營費用的一部分。與前端銷售費相反的是後端銷售費，當投資者出售投資時，通過從利潤或本金中扣除來支付。還有其他類型的基金銷售費，包括按比例收費，這種費用收取持續的年度費用。
