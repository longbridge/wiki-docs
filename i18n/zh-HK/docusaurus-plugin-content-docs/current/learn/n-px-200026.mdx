# N-PX

<p>N-PX 基金招募説明書：投資公司提交的招募説明書，包含基金的營銷材料和投資者需要了解的關鍵信息。</p>

<AIContent content={`<p><strong>定義：</strong><br/>N-PX 是指投資公司提交的基金招募説明書，包含基金的營銷材料和投資者需要了解的關鍵信息。N-PX 文件通常用於披露基金的投票記錄和其他重要信息，幫助投資者做出明智的投資決策。</p><p><strong>起源：</strong><br/>N-PX 文件的起源可以追溯到 2003 年，當時美國證券交易委員會（SEC）要求所有管理公開交易基金的投資公司提交這些文件，以提高透明度和保護投資者利益。</p><p><strong>類別與特點：</strong><br/>1. <strong>投票記錄：</strong> N-PX 文件詳細列出了基金在股東會議上的投票記錄，包括每個議題的投票結果和基金管理層的投票建議。<br/>2. <strong>投資策略：</strong> N-PX 文件還可能包含基金的投資策略和目標，幫助投資者瞭解基金的運作方式。<br/>3. <strong>費用和費用結構：</strong> N-PX 文件通常會披露基金的費用和費用結構，幫助投資者評估投資成本。</p><p><strong>具體案例：</strong><br/>1. <strong>案例一：</strong> 某投資公司提交的 N-PX 文件顯示，該公司在某科技公司的股東會議上投票反對一項高管薪酬計劃，理由是該計劃不符合股東的長期利益。<br/>2. <strong>案例二：</strong> 另一家基金公司在其 N-PX 文件中披露了其對環境、社會和治理（ESG）議題的投票記錄，顯示其在多個公司股東會議上支持了與可持續發展相關的提案。</p><p><strong>常見問題：</strong><br/>1. <strong>為什麼 N-PX 文件重要？</strong> N-PX 文件提供了基金的投票記錄和其他關鍵信息，幫助投資者瞭解基金的運作和決策過程。<br/>2. <strong>如何獲取 N-PX 文件？</strong> 投資者可以通過基金公司的官方網站或美國證券交易委員會（SEC）的 EDGAR 數據庫獲取 N-PX 文件。</p>`} id={200026} />
