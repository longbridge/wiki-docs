# 比特幣悲觀指數

<ArticleMeta id={102280} updatedAt={'2024-09-12 16:59:39'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>比特幣悲觀指數（Bitcoin Fear Index）是一種用於衡量比特幣市場情緒的指標，旨在反映市場參與者對比特幣價格波動的擔憂和恐懼程度。該指數通常基於多種因素計算，包括市場波動性、交易量、社交媒體情緒、市場調查和其他相關數據。比特幣悲觀指數的取值範圍從 0 到 100，高值表示市場情緒較為負面，投資者對比特幣價格下跌的擔憂加劇；低值則表示市場情緒較為樂觀，投資者對比特幣價格上漲的信心增強。該指數幫助投資者理解市場情緒，從而做出更明智的投資決策。</p>
