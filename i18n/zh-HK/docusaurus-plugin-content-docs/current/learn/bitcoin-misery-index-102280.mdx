# 比特幣悲觀指數

<ArticleMeta id={102280} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

比特幣悲觀指數（BMI）是比特幣價格行情的一種度量標準。它的取值範圍從 0 到 100，並利用反常經濟指標，即機會移動與常規指標相反的情況下。它結合了多種不同的市場因素，如勝率和波動性。
