# 超額回報

<ArticleMeta id={100300} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

超額回報是指超過基準回報的回報。超額回報取決於指定的投資回報比較進行分析。一些最基本的回報比較包括無風險利率和與被分析投資具有相似風險水平的基準。
