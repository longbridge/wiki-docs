# 實現收益率

<ArticleMeta id={101608} updatedAt={'2024-09-13 10:34:21'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

實現收益率是指投資持有期間實際獲得的回報。它可能包括股息、利息支付和其他現金分配。術語 “實現收益率” 可以適用於在到期日之前出售的債券或支付股息的證券。一般來説，債券的實現收益率包括持有期間收到的票息支付，加上或減去原始投資價值的變化，以年為基礎計算。

<AIContent content={`<h2>實現收益率</h2><h3>定義</h3><p>實現收益率是指投資者在持有期間實際獲得的回報。這些回報可能包括股息、利息支付和其他現金分配。實現收益率通常用於描述在到期日之前出售的債券或支付股息的證券的收益情況。對於債券來説，實現收益率包括持有期間收到的票息支付，加上或減去原始投資價值的變化，並以年為基礎進行計算。</p><h3>起源</h3><p>實現收益率的概念起源於金融市場的發展，特別是債券市場。隨着投資者對實際收益的關注增加，這一概念逐漸被廣泛應用。關鍵的時間點包括 20 世紀初期金融市場的規範化和現代投資理論的發展。</p><h3>類別與特點</h3><p>實現收益率可以分為以下幾類：</p><ul><li><strong>債券實現收益率：</strong>包括票息支付和資本增值或損失。</li><li><strong>股票實現收益率：</strong>包括股息支付和股票價格的變化。</li></ul><p>特點：</p><ul><li>反映實際獲得的回報，而非預期收益。</li><li>考慮了持有期間的所有現金流。</li><li>以年為基礎進行計算，便於不同投資的比較。</li></ul><h3>具體案例</h3><p><strong>案例 1：</strong>假設投資者購買了一隻面值為 1000 元的債券，票面利率為 5%，持有兩年後以 1100 元的價格賣出。在持有期間，投資者每年收到 50 元的票息支付。實現收益率計算如下：</p><p>實現收益率 = [(50 + 50 + (1100 - 1000)) / 1000] / 2 = 10%</p><p><strong>案例 2：</strong>投資者購買了一隻股票，初始價格為 50 元，持有三年後以 70 元的價格賣出。在持有期間，每年收到 2 元的股息支付。實現收益率計算如下：</p><p>實現收益率 = [(2 + 2 + 2 + (70 - 50)) / 50] / 3 = 16%</p><h3>常見問題</h3><p><strong>Q1:</strong> 實現收益率與預期收益率有何不同？<br/><strong>A1:</strong> 實現收益率是實際獲得的回報，而預期收益率是投資前的估計回報。</p><p><strong>Q2:</strong> 實現收益率是否考慮了通貨膨脹？<br/><strong>A2:</strong> 實現收益率通常不考慮通貨膨脹，需要投資者自行調整。</p>`} id={101608} />
