# 其他流動負債

<ArticleMeta id={102075} updatedAt={'2023-09-25 22:59:00'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>其他流動負債（Other Current Liabilities, OCL）是指企業在一個會計期間內（通常為一年或一個營業週期內）需要償還的各種短期負債，除了傳統流動負債（如應付賬款和短期借款）以外的負債。這類負債在企業的資產負債表中列示，反映了企業在短期內需要履行的財務義務。</p><p>主要特點包括：</p><ol><li><strong>短期償還</strong>：其他流動負債通常需要在一年內或一個營業週期內償還。</li><li><strong>多樣性</strong>：包括多種類型的短期負債，具體內容因企業的業務性質和財務安排而異。</li><li><strong>流動性</strong>：這些負債具有較高的流動性，企業需要確保有足夠的現金流來履行這些義務。</li><li><strong>財務健康</strong>：反映企業短期財務健康狀況，過多的流動負債可能影響企業的流動性和償債能力。</li></ol><p>其他流動負債的例子：</p><ol><li><strong>應付工資</strong>：企業需要支付給員工的未付工資。</li><li><strong>應付税款</strong>：企業應繳納但尚未支付的各種税款，如增值税、所得税等。</li><li><strong>預收收入</strong>：企業已收到但尚未提供相關商品或服務的預收款項。</li><li><strong>應付費用</strong>：企業已經發生但尚未支付的費用，如租金、保險費等。</li><li><strong>應付股利</strong>：企業已經宣告但尚未支付的股利。</li></ol><p>應用示例： 假設一家制造公司在其資產負債表中列示了以下其他流動負債：</p><ul><li>應付工資：$10,000</li><li>應付税款：$5,000</li><li>預收收入：$8,000</li><li>應付租金：$3,000</li><li>應付保險費：$2,000</li></ul><p>這些項目合計為$28,000，表示公司在未來一年內需要償還的其他流動負債總額。</p>
