# 多資產類

<ArticleMeta id={100406} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

多資產類別，又稱多資產類別或多資產基金，是作為投資的一種資產類別的組合。多資產類投資包含多個資產類別，從而創建了一組或資產組合。權重和類別的類型根據個人投資者而異。
