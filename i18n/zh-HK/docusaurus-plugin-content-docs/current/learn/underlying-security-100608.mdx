# 基礎證券

<ArticleMeta id={100608} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

基礎證券指衍生工具，如期貨、ETF 和期權所基於的股票或債券。它是衍生品價值的主要組成部分。
