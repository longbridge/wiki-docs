# 原始發行折扣

<ArticleMeta id={102206} updatedAt={'2024-09-12 02:27:48'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>原始發行折價（Original Issue Discount, OID）是指債券或其他債務工具在發行時以低於其面值的價格出售的情況。OID 代表投資者在債券到期時能夠獲得的額外收益，等於債券面值減去發行價格。該額外收益通常被視為利息收入，並在債券的持有期內逐漸計入投資者的應税收入。</p><p>原始發行折價的主要特點包括：</p><p>低於面值發行：債券以低於其面值的價格發行，投資者在購買時即獲得折扣。<br/>税務處理：OID 被視為利息收入，需在債券持有期間逐步納税，而不是在債券到期時一次性納税。<br/>增加收益：通過原始發行折價，投資者能夠在債券到期時獲得面值與發行價之間的差額作為額外收益。<br/>定價策略：發行人可以通過 OID 吸引更多投資者，尤其是在市場利率較高的情況下，使得債券更具吸引力。<br/>原始發行折價的計算示例：<br/>假設某公司發行一張面值為 1000 美元的債券，但發行價格為 950 美元，持有期為 5 年。該債券的原始發行折價為：<br/>OID = 面值−發行價 = 1000−950 = 50 美元</p><p>在税務處理方面，假設這 50 美元的 OID 需要在債券的 5 年持有期內逐年納税，則每年的應税 OID 收入為：<br/>50 美元/5 年 = 10 美元/年</p>
