# 基本每股虧損

<ArticleMeta id={104557} updatedAt={'2023-10-24 05:20:16'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

基本每股虧損是指企業每股普通股在一定時期內所產生的淨虧損金額，是衡量公司經營業績的重要指標之一。基本每股虧損越小，表明企業盈利能力越強。

<AIContent content={`<p><strong>定義：</strong>基本每股虧損（Basic Loss Per Share, 簡稱 LPS）是指企業每股普通股在一定時期內所產生的淨虧損金額。它是衡量公司經營業績的重要指標之一。基本每股虧損越小，表明企業盈利能力越強。</p><p><strong>起源：</strong>基本每股虧損的概念源自於會計和財務報表分析，最早在 20 世紀中期被廣泛應用於企業財務報告中。隨着資本市場的發展，投資者和分析師越來越重視這一指標，以評估公司的盈利能力和財務健康狀況。</p><p><strong>類別與特點：</strong>基本每股虧損主要有以下幾個特點：<ul><li>計算簡單：基本每股虧損的計算公式為：<br/>基本每股虧損 = 淨虧損 / 普通股加權平均股數</li><li>反映公司盈利能力：基本每股虧損越小，表明公司盈利能力越強，反之則表明公司可能面臨財務困境。</li><li>時間敏感：該指標通常按季度或年度計算，能夠及時反映公司在特定時期內的經營狀況。</li></ul></p><p><strong>具體案例：</strong><ol><li>案例一：某公司在 2023 年第一季度的淨虧損為 1000 萬元，普通股加權平均股數為 500 萬股，則該季度的基本每股虧損為：<br/>基本每股虧損 = 1000 萬元 / 500 萬股 = 2 元/股</li><li>案例二：另一家公司在 2023 年全年淨虧損為 2000 萬元，普通股加權平均股數為 1000 萬股，則該年度的基本每股虧損為：<br/>基本每股虧損 = 2000 萬元 / 1000 萬股 = 2 元/股</li></ol></p><p><strong>常見問題：</strong><ul><li>為什麼基本每股虧損是負數？<br/>因為它反映的是淨虧損，虧損金額為負數，因此基本每股虧損也為負數。</li><li>基本每股虧損和稀釋每股虧損有什麼區別？<br/>基本每股虧損只考慮普通股，而稀釋每股虧損還考慮了潛在的稀釋性證券（如可轉換債券、期權等）。</li></ul></p>`} id={104557} />
