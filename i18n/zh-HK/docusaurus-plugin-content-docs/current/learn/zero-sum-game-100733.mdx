# 零和遊戲

<ArticleMeta id={100733} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

零和遊戲是指一種情況，經常在博弈論中引用，其中一個人的收益等同於另一個人的損失，因此財富或利益的淨變化為零。零和遊戲可以有兩個參與者，也可以有數百萬的參與者。在金融市場上，期權和期貨是零和遊戲的例子，不包括交易成本。對於每個在合約上獲利的人，都有一個相對方會遭受損失。
