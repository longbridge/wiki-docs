# 總回報率

<ArticleMeta id={101265} updatedAt={'2023-09-26 13:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

總回報率是在扣除任何費用、佣金或支出之前的投資總回報率。總回報率是在特定時間段內進行報價的，如一個月、一個季度或一年。這可以與淨回報率相對比，淨回報率扣除費用和成本，以提供更真實的回報率測量。

<AIContent content={`<p><strong>定義：</strong>總回報率（Total Return）是指在扣除任何費用、佣金或支出之前的投資總回報率。它通常在特定時間段內進行計算，如一個月、一個季度或一年。總回報率包括資本增值和任何分紅或利息收入。</p><p><strong>起源：</strong>總回報率的概念起源於現代投資理論的發展，特別是在 20 世紀中期，隨着金融市場的成熟和投資工具的多樣化，投資者開始關注不僅僅是資本增值，還包括其他形式的收益，如股息和利息。</p><p><strong>類別與特點：</strong>總回報率可以分為以下幾類：<ul><li><strong>資本增值：</strong>投資資產價格的上漲部分。</li><li><strong>股息收入：</strong>股票投資中公司分配給股東的利潤部分。</li><li><strong>利息收入：</strong>債券或其他固定收益投資的利息部分。</li></ul>總回報率的特點是全面反映投資的所有收益來源，適用於評估投資的整體表現。</p><p><strong>具體案例：</strong><ol><li>假設你在 2023 年初投資了 10000 元購買某公司的股票，到 2023 年底股票價格上漲到 12000 元，同時你還收到了 500 元的股息。那麼，總回報率計算如下：<br/>總回報率 = [(12000 - 10000) + 500] / 10000 = 0.25 或 25%</li><li>你購買了一隻債券，面值為 10000 元，年利率為 5%，一年後你收到了 500 元的利息，同時債券價格上漲到 10500 元。那麼，總回報率計算如下：<br/>總回報率 = [(10500 - 10000) + 500] / 10000 = 0.10 或 10%</li></ol></p><p><strong>常見問題：</strong><ul><li><strong>總回報率和淨回報率有什麼區別？</strong>總回報率不扣除任何費用和成本，而淨回報率則扣除了這些費用，提供了更真實的回報率測量。</li><li><strong>總回報率是否適用於所有投資類型？</strong>是的，總回報率可以應用於股票、債券、基金等各種投資類型。</li></ul></p>`} id={101265} />
