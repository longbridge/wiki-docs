# 賣空比率

<ArticleMeta id={100689} updatedAt={'2023-09-25 22:58:51'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>賣空比率（Short Interest Ratio, SIR）是衡量市場上賣空活動程度的一個指標。它計算的是某一特定股票的未平倉空頭頭寸數量與該股票平均每日交易量的比值。具體公式為：賣空比率 = 未平倉空頭頭寸數量 / 平均每日交易量。高賣空比率通常表明市場對該股票的看空情緒較強，可能預示着未來的價格下跌。然而，過高的賣空比率也可能導致空頭回補（即空頭投資者買回股票以平倉）的風險，從而推動股票價格上漲。投資者可以通過賣空比率來判斷市場情緒和潛在的價格波動。</p><p>&nbsp;</p>
