# BOLL 值

<ArticleMeta id={104168} updatedAt={'2023-10-18 20:20:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

BOLL 指標是一種基於移動平均線的技術分析指標，它使用中軌線、上軌線和下軌線來顯示股價的壓力區間和支撐區間，從而幫助投資者判斷股價的走勢方向。
