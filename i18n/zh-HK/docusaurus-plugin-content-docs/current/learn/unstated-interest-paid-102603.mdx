# 未披露利息支付額

<ArticleMeta id={102603} updatedAt={'2024-09-12 15:49:10'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

未披露利息支付額是指國內税務機關 (IRS) 假定已支付給按分期付款方式出售商品的賣方的款項。在某些情況下，如果您按分期付款方式出售商品但幾乎沒有向客户收取利息，則必須計算未披露利息。由於利息收入有時必須與其他類型的收入區分對待，因此可能需要估計分期付款中實際利息收入的部分。
