# 美國存託憑證

<ArticleMeta id={100228} updatedAt={'2024-09-13 11:44:58'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>美國存託憑證（ADR）是一種金融工具，代表了在美國市場上市交易的外國公司的股票。ADR 由美國銀行發行，並在美國證券交易所或場外市場交易。每一份 ADR 通常代表一定數量的外國股票。通過 ADR，美國投資者可以方便地投資於外國公司，而無需直接購買外國股票。ADR 不僅為外國公司提供了進入美國資本市場的途徑，還為美國投資者提供了更多的投資選擇。</p>

<AIContent content={`<p><strong>定義：</strong>美國存託憑證（ADR）是一種金融工具，代表了在美國市場上市交易的外國公司的股票。ADR 由美國銀行發行，並在美國證券交易所或場外市場交易。每一份 ADR 通常代表一定數量的外國股票。通過 ADR，美國投資者可以方便地投資於外國公司，而無需直接購買外國股票。ADR 不僅為外國公司提供了進入美國資本市場的途徑，還為美國投資者提供了更多的投資選擇。</p><p><strong>起源：</strong>美國存託憑證的概念最早出現在 1927 年，當時 J.P.摩根銀行為了幫助英國的 Selfridges 百貨公司在美國市場籌集資金，首次發行了 ADR。此後，ADR 逐漸成為外國公司進入美國資本市場的重要工具。</p><p><strong>類別與特點：</strong>ADR 主要分為三類：一級 ADR、二級 ADR 和三級 ADR。<ul><li><strong>一級 ADR：</strong>這種類型的 ADR 在美國場外市場交易，信息披露要求較低，適合那些不打算在美國主要證券交易所上市的公司。</li><li><strong>二級 ADR：</strong>這種類型的 ADR 在美國主要證券交易所（如紐約證券交易所或納斯達克）上市，信息披露要求較高，適合那些希望在美國市場獲得更多曝光的公司。</li><li><strong>三級 ADR：</strong>這種類型的 ADR 不僅在美國主要證券交易所上市，還通過公開發行籌集資金，信息披露要求最高，適合那些希望在美國市場大規模籌資的公司。</li></ul></p><p><strong>具體案例：</strong><ul><li><strong>案例一：</strong>阿里巴巴集團（Alibaba Group）在 2014 年通過發行三級 ADR 在紐約證券交易所上市，成為當時全球最大的 IPO 之一。通過 ADR，阿里巴巴成功吸引了大量美國投資者的關注和資金。</li><li><strong>案例二：</strong>瑞士製藥公司諾華（Novartis）通過發行二級 ADR 在紐約證券交易所上市，使得美國投資者能夠方便地投資於這家歐洲製藥巨頭。</li></ul></p><p><strong>常見問題：</strong><ul><li><strong>問題一：</strong>ADR 的交易費用如何？<br/><strong>解答：</strong>ADR 的交易費用包括交易佣金、存託銀行費用和外匯轉換費用。投資者應在交易前瞭解這些費用。</li><li><strong>問題二：</strong>ADR 的股息如何分配？<br/><strong>解答：</strong>ADR 的股息通常以美元支付，但需要扣除相關税費和存託銀行的管理費用。</li></ul></p>`} id={100228} />
