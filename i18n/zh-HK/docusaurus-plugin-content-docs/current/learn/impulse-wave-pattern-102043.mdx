# 脈衝波動模式

<ArticleMeta id={102043} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

脈衝波動模式是指與基礎趨勢的主要方向相符的金融資產價格強勁波動的指示。脈衝波動可以指上升趨勢中的上升運動或下降趨勢中的下降運動。這個術語經常被艾略特波動理論的信奉者使用，該理論是一種分析和預測金融市場價格變動的方法。
