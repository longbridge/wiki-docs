# 變動成本

<ArticleMeta id={100266} updatedAt={'2023-09-25 22:58:47'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>變動成本是指隨着生產或銷售數量的變化而變化的成本。這些成本直接與生產活動相關，因此當生產量增加時，變動成本也會增加；反之，當生產量減少時，變動成本也會減少。常見的變動成本包括原材料成本、直接勞動力成本、生產用電費用等。變動成本與固定成本相對，後者在短期內不隨生產或銷售數量的變化而變化。瞭解變動成本對於企業進行成本控制和定價策略非常重要。</p>
