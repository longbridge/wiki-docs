# 賬户號碼

<ArticleMeta id={101184} updatedAt={'2023-09-26 13:58:54'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

在共同基金中，賬户號碼是唯一標識您與基金的賬户的號碼。與銀行賬户號碼類似，賬户號碼可用作唯一確定基金投資者的方式。賬户號碼還記錄了每位投資者向基金投入的資金金額、其交易歷史記錄和聯繫方式。賬户號碼還可用於標識日記賬項或土地包裹。不同的基金公司和類似賬户號碼的提供商都會使用略微不同的方法來創建一個數值。
