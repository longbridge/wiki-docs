# 委託 - 代理問題

<ArticleMeta id={102143} updatedAt={'2024-09-12 18:02:28'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>委託 - 代理問題（Principal-Agent Problem）是指在一個組織或交易中，委託人（如股東或老闆）與代理人（如經理或員工）之間由於利益不一致而導致的衝突。代理人負責管理或執行委託人的事務，但由於代理人可能追求自身利益最大化而不是委託人的最佳利益，可能會導致道德風險和逆向選擇等問題。</p><p>主要特點包括：</p><p>利益不一致：委託人和代理人的目標和利益可能不完全一致。<br/>信息不對稱：代理人通常擁有比委託人更多的信息，導致信息不對稱問題。<br/>道德風險：代理人可能利用其信息優勢或職位便利，追求自身利益而不是委託人的最佳利益。<br/>逆向選擇：在信息不對稱下，委託人難以選擇合適的代理人，可能導致劣質代理人被選擇。<br/>委託 - 代理問題的應用示例：<br/>假設一家公司的股東（委託人）希望公司管理層（代理人）努力增加公司利潤，但管理層可能更關注個人薪酬和福利。由於股東無法實時監督所有管理決策，管理層可能會做出一些有利於自身利益但不一定有利於公司長期發展的決策，如過度消費公司資源或追求短期收益。</p><p>解決委託 - 代理問題的方法：</p><p>激勵機制：設計合理的激勵機制，使代理人的利益與委託人的利益一致，例如績效薪酬和股權激勵。<br/>監督與控制：加強對代理人的監督和控制，減少信息不對稱和道德風險。<br/>信息披露：增加透明度，要求代理人定期披露關鍵決策和財務信息，減少信息不對稱。<br/>契約安排：通過詳細的契約安排，明確代理人的職責和行為規範，防止代理人行為偏離委託人利益。</p>
