# 滑行路徑

<ArticleMeta id={100571} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

滑行路徑是指根據距離目標日期的年數定義目標日期基金的資產配置組合的公式。滑行路徑創建了一個資產配置，通常在基金接近目標日期時變得更加保守 (即包含更多的固定收益資產和較少的股票資產)。
