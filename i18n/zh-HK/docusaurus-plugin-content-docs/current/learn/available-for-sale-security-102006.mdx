# 可供出售證券

<ArticleMeta id={102006} updatedAt={'2023-09-26 13:59:00'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

可供出售證券 (AFS) 是指以出售意圖購買的債務或權益證券。在達到到期日之前出售，或在沒有到期日的情況下長期持有。會計準則要求公司將債務或權益證券的任何投資在購買時分類為持有到期日、持有交易或可供出售。可供出售證券以公允價值報告；會計期間價值變動計入資產負債表的股東權益部分積累其他綜合收益中。

<AIContent content={`<p><strong>定義：</strong>可供出售證券（Available-for-Sale Securities，簡稱 AFS）是指公司購買的債務或權益證券，這些證券可能在到期日之前出售，或者在沒有到期日的情況下長期持有。根據會計準則，公司需要在購買時將這些證券分類為持有到期日、持有交易或可供出售。可供出售證券以公允價值報告，其價值變動計入資產負債表的股東權益部分的積累其他綜合收益中。</p><p><strong>起源：</strong>可供出售證券的概念源於會計準則的演變，特別是為了更好地反映公司投資的真實價值和風險。20 世紀末，隨着金融市場的複雜化和全球化，會計準則逐漸引入了對金融工具分類和公允價值計量的要求。</p><p><strong>類別與特點：</strong>可供出售證券主要分為兩類：債務證券和權益證券。<ul><li><strong>債務證券：</strong>包括公司債券、政府債券等，通常有固定的到期日和利息支付。</li><li><strong>權益證券：</strong>包括公司股票等，沒有固定的到期日，持有者可以通過股息和資本增值獲利。</li></ul>可供出售證券的主要特點是其公允價值變動計入其他綜合收益，而不是直接影響當期損益。</p><p><strong>具體案例：</strong><ul><li><strong>案例一：</strong>某公司購買了一批政府債券，計劃在市場條件有利時出售。這些債券被分類為可供出售證券。在會計期間，這些債券的市場價值上升，公司將增值部分計入資產負債表的其他綜合收益中。</li><li><strong>案例二：</strong>某公司購買了一些上市公司的股票，作為長期投資。這些股票被分類為可供出售證券。在會計期間，這些股票的市場價值下降，公司將減值部分計入資產負債表的其他綜合收益中。</li></ul></p><p><strong>常見問題：</strong><ul><li><strong>問題一：</strong>可供出售證券的公允價值變動如何影響財務報表？<br/><strong>解答：</strong>公允價值變動計入資產負債表的其他綜合收益中，不直接影響當期損益。</li><li><strong>問題二：</strong>可供出售證券與持有交易證券有何區別？<br/><strong>解答：</strong>持有交易證券的公允價值變動直接計入當期損益，而可供出售證券的公允價值變動計入其他綜合收益。</li></ul></p>`} id={102006} />
