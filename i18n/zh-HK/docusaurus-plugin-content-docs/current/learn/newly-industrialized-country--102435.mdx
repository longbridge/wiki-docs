# 新興工業化國家

<ArticleMeta id={102435} updatedAt={'2024-09-12 01:24:08'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

新興工業化國家（NIC）是政治學家和經濟學家用來描述一個國家的經濟發展水平介於發展中國家和高度發達國家之間的術語。這些國家已經擺脱了以農業為基礎的經濟體制，轉向了更加工業化、城市化的經濟體制。專家們也將它們稱為 “新興工業化經濟體” 或 “先進發展中國家”。
