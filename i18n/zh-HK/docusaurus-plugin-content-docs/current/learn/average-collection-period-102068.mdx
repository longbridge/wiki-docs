# 平均收款週期

<ArticleMeta id={102068} updatedAt={'2024-09-12 03:37:50'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>平均收款期（Average Collection Period, ACP）是指企業從銷售商品或服務到收到客户付款所需的平均天數。該指標反映了企業應收賬款的管理效率和客户支付的及時性。平均收款期越短，説明企業能夠更快地回收現金，提高資金的週轉效率；平均收款期越長，則可能表明企業在應收賬款管理或客户信用控制方面存在問題。</p><p>主要特點包括：</p><ol><li><strong>反映收款效率</strong>：衡量企業應收賬款的收款效率和客户支付的及時性。</li><li><strong>財務管理指標</strong>：用於評估企業資金週轉情況和財務管理效果。</li><li><strong>信用控制</strong>：反映企業的信用政策和客户管理效果。</li><li><strong>與現金流相關</strong>：平均收款期越短，企業現金流越穩定；反之，現金流壓力可能增大。</li></ol><p>平均收款期的計算公式：&nbsp;</p><p>平均收款期 = (應收賬款總額/年銷售收入)×365</p><p>或&nbsp;</p><p>平均收款期 = 應收賬款總額/日銷售收入)</p><p>應用示例： 假設某企業在年度結束時的應收賬款總額為$100,000，年度銷售收入為$1,000,000，則其平均收款期計算如下：&nbsp;</p><p>平均收款期 = (100,000/1,000,000)×365=36.5 天</p><p>這意味着該企業從銷售到收款平均需要 36.5 天。</p>
