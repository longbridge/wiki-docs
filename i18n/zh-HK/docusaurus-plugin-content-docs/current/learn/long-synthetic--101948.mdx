合成看跌期權是一種期權策略，將空頭持倉與該股票的認購期權合併，以模擬看跌期權。也稱為合成看跌期權。實質上，持有空頭股票倉位的投資者購買同一股票的平價認購期權，以防止股票價格上漲。合成看跌期權也被稱為婚姻型認購期權或保護型認購期權。
