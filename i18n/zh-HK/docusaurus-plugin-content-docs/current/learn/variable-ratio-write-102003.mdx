# 可變比例寫作

<ArticleMeta id={102003} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

可變比例寫作是一種期權投資策略，要求在持有標的資產的同時，同時寫入多個具有不同行權價格的認購期權。它本質上是一種比例買入 - 寫作策略。交易員的目標是獲得認購期權的權利金。可變比例寫作的利潤潛力有限。該策略最適用於預期波動性較小（特別是在短期內）的股票。
