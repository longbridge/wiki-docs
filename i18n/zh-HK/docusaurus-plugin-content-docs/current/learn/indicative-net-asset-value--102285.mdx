# 參考淨資產價值

<ArticleMeta id={102285} updatedAt={'2024-09-12 01:57:25'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>參考淨資產價值（Net Asset Value, NAV）是指公司或基金在扣除所有負債後的資產總價值。NAV 通常用於評估共同基金、ETF 或投資公司的每股淨值，是投資者衡量其投資價值的重要指標。計算公式為：</p><figure class="image image_resized" style={{width:"33.02%"}}><img src="https://pub.lbkrs.com/social/2024/0/KiriEfk5P12dpHzFL853owecZQMwxR3W.jpg" /></figure><p>其中，總資產包括現金、股票、債券和其他投資，總負債包括所有的短期和長期債務。參考淨資產價值每日計算一次，反映了基金或公司的當前市場價值，幫助投資者做出買入或賣出的決策。</p>
