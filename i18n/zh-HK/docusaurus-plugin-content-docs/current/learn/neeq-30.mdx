# 全國股轉系統

<p>全國股轉系統（NEEQ），俗稱 “新三板”，是中國為中小企業提供股票發行、交易和融資服務的全國性證券交易場所。新三板主要服務於創新型、創業型、成長型中小企業，旨在為這些企業提供一個相對便捷的融資渠道和流動性支持。</p>
