# 協方差

<ArticleMeta id={100184} updatedAt={'2024-09-13 11:46:40'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>協方差是一種度量兩個資產的收益率之間方向關係的統計工具。正的協方差表示資產收益的同向變動，&nbsp;負的協方差表示資產收益的反向變動。</p><p>協方差是通過分析收益率 (與預期收益的標準差) 或者將兩個隨機變量的相關性乘以各自變量的標準差來計算的。</p>

<AIContent content={`<h2>定義</h2><p>協方差是一種度量兩個資產的收益率之間方向關係的統計工具。正的協方差表示資產收益的同向變動，負的協方差表示資產收益的反向變動。協方差是通過分析收益率 (與預期收益的標準差) 或者將兩個隨機變量的相關性乘以各自變量的標準差來計算的。</p><h2>起源</h2><p>協方差的概念最早可以追溯到 19 世紀，由卡爾·皮爾遜（Karl Pearson）等統計學家提出。它最初用於生物統計學，後來逐漸被引入金融領域，用於分析資產之間的關係。</p><h2>類別與特點</h2><p>協方差主要分為兩類：樣本協方差和總體協方差。樣本協方差用於有限樣本數據的分析，而總體協方差則用於整個數據集的分析。樣本協方差的計算公式為：</p><p>$$Cov(X, Y) = \frac{1}{n-1} \sum_{i=1}^{n} (X_i - \bar{X})(Y_i - \bar{Y})$$</p><p>總體協方差的計算公式為：</p><p>$$Cov(X, Y) = \frac{1}{N} \sum_{i=1}^{N} (X_i - \mu_X)(Y_i - \mu_Y)$$</p><p>其中，$X_i$和$Y_i$分別是兩個變量的觀測值，$\bar{X}$和$\bar{Y}$是樣本均值，$\mu_X$和$\mu_Y$是總體均值，$n$是樣本數量，$N$是總體數量。</p><h2>具體案例</h2><p>案例 1：假設你投資了兩隻股票 A 和 B。通過計算它們的協方差，你發現協方差為正，説明這兩隻股票的收益率通常是同向變動的。如果 A 股票上漲，B 股票也有可能上漲。</p><p>案例 2：假設你投資了股票 C 和債券 D。通過計算它們的協方差，你發現協方差為負，説明這兩種資產的收益率通常是反向變動的。如果 C 股票上漲，D 債券可能會下跌。</p><h2>常見問題</h2><p>1. 協方差為零是否意味着兩個資產完全不相關？</p><p>不一定。協方差為零表示兩個資產的收益率沒有線性關係，但它們可能存在非線性關係。</p><p>2. 協方差和相關係數有什麼區別？</p><p>協方差度量的是兩個變量之間的方向關係，而相關係數則標準化了協方差，範圍在-1 到 1 之間，更易於解釋。</p>`} id={100184} />
