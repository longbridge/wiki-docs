# 清算價值

<ArticleMeta id={100846} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

清算價值是指一家公司如果倒閉並出售其資產，其物質資產的淨值。清算價值包括公司的房地產、設備、固定資產和庫存。無形資產不包括在一家公司的清算價值中。
