# 年回報率

<ArticleMeta id={100793} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

年回報率是指投資在一段時間內所提供的回報，以時間加權的年百分比來表示。回報的來源可以包括股息、資本回報和資本增值。年回報率的計算是與投資的初始金額相比，代表的是幾何平均數而不是簡單平均數。
