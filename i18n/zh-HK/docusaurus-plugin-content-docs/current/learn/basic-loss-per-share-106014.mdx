# 每股基本虧損

<ArticleMeta id={106014} updatedAt={'2023-11-10 20:21:36'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

每股基本虧損是指公司在一定期間內，每股普通股權益所承擔的淨利潤虧損。它是衡量公司盈利能力的重要指標之一。每股基本虧損越小，説明公司虧損能力越強，投資風險也越大。
