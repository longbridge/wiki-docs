# 住宅按揭支持證券

<ArticleMeta id={102629} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

住宅按揭支持證券（RMBS）是以住房貸款利息作為擔保的債務性資產。由於個人或家庭住宅的所有權需求很高，所以抵押貸款和住房貸款的違約率相對較低，利率較高。通過將這些貸款進行分散池化，可以降低投資者的風險。
