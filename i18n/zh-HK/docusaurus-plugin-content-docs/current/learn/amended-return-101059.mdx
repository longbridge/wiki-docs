# 修正申報

<ArticleMeta id={101059} updatedAt={'2023-09-25 22:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

修正申報是指為了糾正上一年度税務申報錯誤而提交的表單。修正申報可以糾正錯誤並要求更有利的税務狀態，例如退款。例如，在錯誤報告的收入或税收抵免情況下，可以選擇提交修正申報。然而，數學錯誤不需要修正，因為在處理税務申報時，税務機關會自動糾正此類錯誤。
