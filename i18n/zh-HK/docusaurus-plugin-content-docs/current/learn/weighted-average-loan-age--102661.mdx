# 加權平均貸款年齡 

<ArticleMeta id={102661} updatedAt={'2023-09-25 22:59:02'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>加權平均貸款年齡 (WALA) 是衡量按其餘餘額或貸款名義價值對每個抵押支持證券池中貸款年齡的平均數的指標，權重基於在每個到期日的每筆貸款的美元金額與池子的總金額的比例。</p>

<AIContent content={`<p><strong>定義：</strong>加權平均貸款年齡 (WALA) 是衡量按其餘餘額或貸款名義價值對每個抵押支持證券池中貸款年齡的平均數的指標，權重基於在每個到期日的每筆貸款的美元金額與池子的總金額的比例。</p><p><strong>起源：</strong>加權平均貸款年齡的概念起源於抵押支持證券（MBS）市場，旨在幫助投資者評估證券池中貸款的平均年齡。隨着 MBS 市場的發展，WALA 成為衡量貸款池風險和預期收益的重要工具。</p><p><strong>類別與特點：</strong>WALA 主要用於評估抵押支持證券池中的貸款年齡。其特點包括：1. 反映貸款池的整體年齡結構；2. 幫助投資者評估貸款池的風險和預期收益；3. 通過加權計算，提供更準確的貸款年齡分佈信息。WALA 的應用場景主要在 MBS 市場，投資者可以通過 WALA 瞭解貸款池的平均年齡，從而做出更明智的投資決策。</p><p><strong>具體案例：</strong>案例 1：某 MBS 池包含三筆貸款，金額分別為$100,000、$200,000 和$300,000，貸款年齡分別為 2 年、3 年和 5 年。WALA 計算如下：WALA = (100,000*2 + 200,000*3 + 300,000*5) / (100,000 + 200,000 + 300,000) = 4 年。案例 2：另一個 MBS 池包含四筆貸款，金額分別為$150,000、$250,000、$350,000 和$250,000，貸款年齡分別為 1 年、2 年、4 年和 6 年。WALA 計算如下：WALA = (150,000*1 + 250,000*2 + 350,000*4 + 250,000*6) / (150,000 + 250,000 + 350,000 + 250,000) = 3.75 年。</p><p><strong>常見問題：</strong>1. 為什麼 WALA 對投資者重要？WALA 幫助投資者瞭解貸款池的平均年齡，從而評估其風險和預期收益。2. WALA 與加權平均貸款期限（WAM）有何不同？WALA 關注的是貸款的實際年齡，而 WAM 關注的是貸款的剩餘期限。</p>`} id={102661} />
