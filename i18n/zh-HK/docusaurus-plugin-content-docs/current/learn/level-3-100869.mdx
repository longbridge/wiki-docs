# 三級報價

<ArticleMeta id={100869} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

Level 3 報價是由交易服務提供的有關證券的定價信息。它包括實時買入價、賣出價、報價大小、最後交易價格、最後交易量、當天高價和當天低價。Level 3 使機構能夠輸入報價、執行訂單和發送信息。由於 Level 3 服務提供了較高水平的市場深度，因此僅限於註冊的納斯達克市場交易商。
