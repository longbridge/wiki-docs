# 名義本金金額

<p><span style={{backgroundColor:"rgb(255,255,255)",color:"rgb(6,6,7)"}}>名義本金金額指的是在金融交易，如利率互換中，用作計算利息支付的基礎金額。這個金額是名義上的，並不涉及實際的本金交換，而是作為利息計算的基準。</span></p><p>名義本金金額通常用於衍生品合約，幫助確定交易雙方的現金流，但並不涉及實際的資金流動。這個概念在風險管理和金融工程中非常重要，因為它允許市場參與者評估和管理與利率變動相關的風險。</p>
