# 按揭貸款

<ArticleMeta id={100236} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

按揭貸款是一種貸款類型，借款人僅需要在一定期限內支付貸款利息。本金可以在指定日期一次性償還，或者分期還款。
