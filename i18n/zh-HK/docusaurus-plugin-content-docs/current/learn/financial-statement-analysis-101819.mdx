# 財務報表分析

<ArticleMeta id={101819} updatedAt={'2023-09-26 13:58:58'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

財務報表分析是為了決策目的而對公司財務報表進行分析的過程。外部利益相關者使用它來了解組織的整體健康狀況，並評估財務績效和業務價值。內部利益相關者將其用作管理財務的監控工具。
