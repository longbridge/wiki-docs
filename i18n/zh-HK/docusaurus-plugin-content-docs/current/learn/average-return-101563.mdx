# 平均收益率

<ArticleMeta id={101563} updatedAt={'2023-09-26 13:58:58'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

平均收益率是在指定的時間段內產生的一系列收益的簡單數學平均值。平均收益率的計算方法與計算任何一組數字的簡單平均值相同。將這些數字相加得到一個總數，然後將總數除以該組數字的數量。
