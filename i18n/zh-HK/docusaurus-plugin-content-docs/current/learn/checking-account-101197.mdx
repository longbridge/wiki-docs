# 支票賬户

<ArticleMeta id={101197} updatedAt={'2024-09-12 19:54:31'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

支票賬户是一種存款賬户，可以方便地進行取款、存款和資金轉賬。也稱為需求賬户或交易賬户，可以使用支票、自動櫃員機 (ATM) 和電子借記等方式進行訪問。它們通常用於儲存短期費用。
