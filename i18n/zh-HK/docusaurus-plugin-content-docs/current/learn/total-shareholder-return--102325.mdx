# 股東總回報率

<ArticleMeta id={102325} updatedAt={'2024-09-12 16:42:16'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>股東總回報率（Total Shareholder Return, TSR）是衡量公司在一定時期內為股東創造的總回報的一個重要指標。它包括股息收入和資本增值兩個部分。具體來説，TSR 是指某一時間段內，股東通過持有公司股票所獲得的總收益，包括股票價格的增值以及股息的再投資收益。TSR 的計算公式為：&nbsp;</p><figure class="image image_resized" style={{width:"41.99%"}}><img src="https://pub.lbkrs.com/social/2024/0/amb16JtR1vdiqZ4NM4EezgwDNj33SYXW.jpg" /></figure><p>其中，P<sub>end</sub>​ 是期末股價，P<sub>begin​</sub> 是期初股價，D 是期間內發放的股息總額。TSR 能夠全面反映公司股東所獲得的實際回報，是投資者評估公司績效的重要指標。</p>
