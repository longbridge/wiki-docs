# 股東總回報率

<ArticleMeta id={102325} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

股東總回報率 (TSR) 是衡量財務績效的指標，它表示投資者從投資中獲得的總金額——具體來説，是股權或股票。為了得到總回報率，通常以百分比表示，TSR 考慮了股票的資本收益和股息; 還可能包括特殊分配、股票分割和權證。無論如何計算，TSR 代表的都是同樣的含義：股票對投資者的總回報。
