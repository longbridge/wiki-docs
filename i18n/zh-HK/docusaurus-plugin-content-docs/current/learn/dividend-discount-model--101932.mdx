# 股息折現模型

<ArticleMeta id={101932} updatedAt={'2023-09-26 13:58:58'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p><span style={{backgroundColor:"rgb(255,255,255)",color:"rgb(6,6,7)"}}>股息折現模型是一種評估股票內在價值的財務方法，該理論認為股票的當前價格應等於其未來所有股息的現值總和。</span></p><p><span style={{backgroundColor:"rgb(255,255,255)",color:"rgb(6,6,7)"}}>該模型通過考慮預期股息支付和適當的折現率來估算股票的公允市場價值。若模型計算出的價值高於市場交易價格，股票可能被視為低估；若低於市場交易價格，則可能被視為高估。</span></p>

<AIContent content={`<p><strong>定義：</strong></p><p>股息折現模型（Dividend Discount Model，簡稱 DDM）是一種評估股票內在價值的財務方法。該理論認為，股票的當前價格應等於其未來所有股息的現值總和。通過考慮預期股息支付和適當的折現率，DDM 可以估算股票的公允市場價值。如果模型計算出的價值高於市場交易價格，股票可能被視為低估；反之，則可能被視為高估。</p><p><strong>起源：</strong></p><p>股息折現模型的概念最早可以追溯到 20 世紀初期，但其正式提出和廣泛應用是在 20 世紀 50 年代。該模型的基礎理論由經濟學家約翰·伯爾·威廉姆斯（John Burr Williams）在其 1940 年的著作《投資價值理論》中詳細闡述。</p><p><strong>類別與特點：</strong></p><p>股息折現模型主要分為三種類型：</p><ul><li><strong>零增長模型：</strong>假設公司未來的股息支付保持不變。適用於那些股息穩定且增長緩慢的公司。</li><li><strong>恆定增長模型（戈登增長模型）：</strong>假設公司未來的股息以一個固定的速率增長。適用於那些股息增長穩定的公司。</li><li><strong>多階段增長模型：</strong>假設公司未來的股息增長率會經歷多個階段的變化。適用於那些股息增長率不穩定的公司。</li></ul><p><strong>具體案例：</strong></p><p>案例 1：假設某公司當前支付的股息為每股 2 元，預期未來每年股息增長率為 5%，投資者要求的折現率為 10%。根據恆定增長模型，股票的內在價值計算如下：</p><p>內在價值 = 2 / (10% - 5%) = 40 元</p><p>如果市場價格低於 40 元，投資者可能認為該股票被低估。</p><p>案例 2：假設另一家公司當前支付的股息為每股 3 元，預期未來 5 年股息增長率為 8%，之後穩定在 3%。投資者要求的折現率為 10%。根據多階段增長模型，股票的內在價值計算如下：</p><p>前 5 年股息現值 = 3 * (1 + 8%) / (1 + 10%) + 3 * (1 + 8%)^2 / (1 + 10%)^2 + ... + 3 * (1 + 8%)^5 / (1 + 10%)^5</p><p>之後的股息現值 = 3 * (1 + 8%)^5 * (1 + 3%) / (10% - 3%) / (1 + 10%)^5</p><p>將兩部分相加，即可得出股票的內在價值。</p><p><strong>常見問題：</strong></p><p>1. <strong>股息不穩定怎麼辦？</strong> 對於股息不穩定的公司，可以使用多階段增長模型來更準確地反映其股息變化。</p><p>2. <strong>如何確定折現率？</strong> 折現率通常根據投資者的要求收益率或市場平均收益率來確定。</p>`} id={101932} />
