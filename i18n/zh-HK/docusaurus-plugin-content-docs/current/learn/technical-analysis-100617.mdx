# 技術分析

<ArticleMeta id={100617} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

技術分析是一種交易學科，通過分析從交易活動中收集的統計趨勢，如價格變動和交易量，來評估投資並識別交易機會。與基本分析不同，基本分析試圖根據業務結果（如銷售額和收益）來評估證券的價值，而技術分析側重於價格和交易量的研究。
