# 債務證券

<ArticleMeta id={101174} updatedAt={'2023-09-26 13:58:54'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

債務證券是一種可以在兩個交易方之間買賣的債務工具，具有基本的定義條款，如名義金額 (借入金額)、利率、到期日和續存期日。債務證券的例子包括政府債券、公司債券、存款證明 (CD)、市政債券或優先股。債務證券也可以以抵押證券的形式出現，例如抵押債務證券 (CDO)、抵押貸款證券 (CMO)、由國家抵押協會 (GNMA) 發行的抵押證券 (MBS) 和零息票證券。
