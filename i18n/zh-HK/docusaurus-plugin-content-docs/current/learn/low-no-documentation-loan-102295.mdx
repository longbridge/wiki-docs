# 低/無證明貸款

<ArticleMeta id={102295} updatedAt={'2024-09-12 01:53:36'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>低/無證明貸款（Low/No Documentation Loan）是指借款人在申請貸款時只需提供很少或不提供收入、資產和就業證明的貸款類型。此類貸款通常用於自僱人士、自由職業者或收入波動較大的人羣，他們可能難以提供傳統貸款所需的全面財務文件。低/無證明貸款的審批標準較為寬鬆，借款人只需提供基本的個人信息和信用報告。這種貸款類型在 2008 年金融危機前非常流行，但由於其高風險性，許多貸款最終違約。現在，這類貸款的發放更加嚴格，貸款利率和費用也較高，以補償貸款機構承擔的更大風險。</p>
