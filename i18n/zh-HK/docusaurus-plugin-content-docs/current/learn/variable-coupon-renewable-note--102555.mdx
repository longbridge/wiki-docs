# 可變息可更新債券

<ArticleMeta id={102555} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

可變息可更新債券（VCR）是一種具有定期重置可變票息的可再生固定收益證券。可再生債券是一種具有每週到期的債務證券，該證券的本金會在每週到期時自動重新投資於新的利率。
