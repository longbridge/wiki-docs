# 全球金融穩定報告

<ArticleMeta id={102582} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

《全球金融穩定報告》（Global Financial Stability Report，簡稱 GFSR）是國際貨幣基金組織（IMF）發佈的半年度報告，評估全球金融市場和新興市場融資的穩定性。每年四月和十月發佈兩次。
