# 實收資本

<ArticleMeta id={100909} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

實收資本是指公司通過出售股票直接向投資者籌集到的資金金額。實收資本是在公司將其股票在一級市場上直接出售給投資者時創建的，通常是通過首次公開發行 (IPO) 來實現。當投資者在二級市場進行買賣時，不會產生額外的實收資本，因為這些交易的收益流向賣方股東而不是發行公司。
