# 無槓桿貝塔

<ArticleMeta id={101369} updatedAt={'2023-09-25 22:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>無槓桿貝塔是衡量公司在沒有任何債務情況下的市場風險的一種指標。它反映了公司自身業務的風險，而不考慮其資本結構的影響。無槓桿貝塔通過剔除公司財務槓桿的影響，使得投資者可以更準確地評估公司的基本風險。</p>

<AIContent content={`<p><strong>定義：</strong>無槓桿貝塔（Unlevered Beta）是衡量公司在沒有任何債務情況下的市場風險的一種指標。它反映了公司自身業務的風險，而不考慮其資本結構的影響。無槓桿貝塔通過剔除公司財務槓桿的影響，使得投資者可以更準確地評估公司的基本風險。</p><p><strong>起源：</strong>無槓桿貝塔的概念源自資本資產定價模型（CAPM），該模型由威廉·夏普（William Sharpe）在 20 世紀 60 年代提出。CAPM 用於評估資產的預期收益率，其中貝塔係數是衡量資產相對於市場波動的風險指標。無槓桿貝塔則進一步剔除了財務槓桿的影響，使得投資者可以更純粹地評估公司的業務風險。</p><p><strong>類別與特點：</strong>無槓桿貝塔主要有以下特點：<ul><li>剔除財務槓桿：無槓桿貝塔不考慮公司的債務，專注於公司的業務風險。</li><li>行業對比：無槓桿貝塔可以用於不同公司的行業對比，因為它排除了資本結構的影響。</li><li>風險評估：無槓桿貝塔幫助投資者更準確地評估公司的基本風險。</li></ul></p><p><strong>具體案例：</strong><ol><li>假設公司 A 的槓桿貝塔為 1.5，債務權益比為 0.5，企業税率為 30%。無槓桿貝塔的計算公式為：<br/>無槓桿貝塔 = 槓桿貝塔 / [1 + (1 - 税率) * 債務權益比]<br/>代入數據：無槓桿貝塔 = 1.5 / [1 + (1 - 0.3) * 0.5] = 1.2</li><li>公司 B 和公司 C 在同一行業，分別有不同的資本結構。通過計算無槓桿貝塔，投資者可以更準確地比較兩家公司在沒有債務情況下的業務風險。</li></ol></p><p><strong>常見問題：</strong><ul><li><strong>問：</strong>無槓桿貝塔和槓桿貝塔有什麼區別？<br/><strong>答：</strong>無槓桿貝塔剔除了公司的債務影響，專注於業務風險；槓桿貝塔則包括了公司的財務槓桿。</li><li><strong>問：</strong>為什麼要使用無槓桿貝塔？<br/><strong>答：</strong>使用無槓桿貝塔可以更準確地評估公司的基本業務風險，特別是在進行行業對比時。</li></ul></p>`} id={101369} />
