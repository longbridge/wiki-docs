# 聯合申報測試

<ArticleMeta id={102025} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

聯合申報測試是 IRS 的一項測試，潛在扶養對象必須通過該測試才能被另一個納税人認為是扶養對象之一。聯合申報測試規定，任何扶養對象不能與配偶一起申報聯合納税，同時仍然可以被他人認為是其它納税人的扶養對象，如父母或監護人。然而，此規則有一個例外。由於申報扶養對象具有價值，IRS 制定了幾項測試，如聯合申報測試，以確保扶養對象不會被重複計數。
