# 收益等價

<ArticleMeta id={100934} updatedAt={'2023-09-26 13:58:51'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

收益等價是指應税證券的利率，以產生與免税證券收益相等的回報，反之亦然。

<AIContent content={`<p><strong>定義：</strong>收益等價（Taxable Equivalent Yield, TEY）是指應税證券的利率，以產生與免税證券收益相等的回報，反之亦然。它幫助投資者比較應税和免税投資的實際收益。</p><p><strong>起源：</strong>收益等價的概念起源於税收政策的變化，特別是在政府發行免税債券時。投資者需要一種方法來比較這些免税債券與應税投資的實際收益，從而做出更明智的投資決策。</p><p><strong>類別與特點：</strong>收益等價主要分為兩類：應税收益等價和免税收益等價。應税收益等價用於計算應税投資的等效收益率，而免税收益等價則用於計算免税投資的等效收益率。其特點是考慮了税收對投資回報的影響，使得不同税收待遇的投資可以進行直接比較。</p><p><strong>具體案例：</strong>案例 1：假設某投資者的税率為 30%，他持有一張年收益率為 5% 的免税市政債券。為了找到等效的應税收益率，他可以使用公式：應税收益等價 = 免税收益率 / (1 - 税率)。在此例中，等效的應税收益率為 5% / (1 - 0.30) = 7.14%。案例 2：另一位投資者持有一張年收益率為 6% 的應税公司債券，税率為 25%。為了找到等效的免税收益率，他可以使用公式：免税收益等價 = 應税收益率 * (1 - 税率)。在此例中，等效的免税收益率為 6% * (1 - 0.25) = 4.5%。</p><p><strong>常見問題：</strong>1. 如何計算收益等價？使用公式：應税收益等價 = 免税收益率 / (1 - 税率) 或 免税收益等價 = 應税收益率 * (1 - 税率)。2. 為什麼要使用收益等價？它幫助投資者比較不同税收待遇的投資回報，從而做出更明智的投資決策。</p>`} id={100934} />
