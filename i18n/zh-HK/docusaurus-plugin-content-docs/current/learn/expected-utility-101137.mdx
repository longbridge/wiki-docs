# 預期效用

<ArticleMeta id={101137} updatedAt={'2023-09-25 22:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>預期效用用於描述在不確定性條件下，個體如何做出選擇。期望效用理論假設個體在面對不同可能結果的決策時，會根據每個結果的效用（即結果對個體的滿意度或價值）和該結果發生的概率，計算出每個決策的期望效用值。然後，個體會選擇期望效用最大的決策。</p>

<AIContent content={`<p><strong>定義：</strong>預期效用用於描述在不確定性條件下，個體如何做出選擇。期望效用理論假設個體在面對不同可能結果的決策時，會根據每個結果的效用（即結果對個體的滿意度或價值）和該結果發生的概率，計算出每個決策的期望效用值。然後，個體會選擇期望效用最大的決策。</p><p><strong>起源：</strong>預期效用理論由丹尼爾·伯努利在 1738 年提出，旨在解決聖彼得堡悖論。伯努利提出，決策者應考慮效用而非金錢的絕對值，因為效用反映了個人對結果的主觀滿意度。</p><p><strong>類別與特點：</strong>預期效用理論主要分為兩類：1. <em>風險中性</em>：個體只關心期望值，不考慮風險。2. <em>風險厭惡</em>：個體更傾向於選擇風險較小的選項，即使期望值較低。<br/>特點：1. 強調主觀滿意度。2. 適用於不確定性決策。3. 依賴概率和效用的計算。</p><p><strong>相似概念對比：</strong>預期效用與<em>前景理論</em>相似，但前景理論更關注決策者在面對損失和收益時的心理反應。</p><p><strong>具體案例：</strong>案例 1：投資者在選擇股票時，會根據每隻股票的預期收益和風險，計算出每隻股票的期望效用，並選擇期望效用最大的股票。<br/>案例 2：保險購買者會根據不同保險方案的賠付金額和發生概率，計算出每個方案的期望效用，並選擇期望效用最大的方案。</p><p><strong>常見問題：</strong>1. 如何計算效用？效用通常通過個人偏好或歷史數據來估計。2. 為什麼有時期望效用理論無法解釋實際行為？因為實際決策中，人們的心理因素和認知偏差可能影響決策。</p>`} id={101137} />
