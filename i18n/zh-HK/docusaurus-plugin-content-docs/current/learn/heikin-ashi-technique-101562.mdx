# 平均蠟燭圖

<ArticleMeta id={101562} updatedAt={'2023-09-26 13:58:58'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

平均蠟燭圖使用平均價格數據創建一個日本蠟燭圖，以過濾掉市場噪音。平均蠟燭圖與標準蠟燭圖具有一些相似之處，但是根據用於創建每根蠟燭的數值而不同。平均蠟燭圖不像標準蠟燭圖一樣使用開盤價、最高價、最低價和收盤價，而是使用基於兩期均值的修正公式。這使得圖表呈現更平滑的外觀，更容易發現趨勢和反轉，但也隱藏了缺口和一些價格數據。

<AIContent content={`<p><strong>定義：</strong>平均蠟燭圖（Heikin-Ashi）是一種技術分析工具，通過使用平均價格數據來創建日本蠟燭圖，以過濾掉市場噪音。與標準蠟燭圖不同，平均蠟燭圖使用基於兩期均值的修正公式來計算每根蠟燭的開盤價、收盤價、最高價和最低價。</p><p><strong>起源：</strong>平均蠟燭圖起源於日本，Heikin-Ashi 在日語中意為 “平均條”。這種圖表類型在 20 世紀被引入西方金融市場，並逐漸被技術分析師所接受和使用。</p><p><strong>類別與特點：</strong>平均蠟燭圖的主要特點是其平滑的外觀，這使得趨勢和反轉更容易被識別。與標準蠟燭圖相比，平均蠟燭圖隱藏了價格缺口和一些細節數據，從而減少了市場噪音。其計算公式如下：<br/>1. 開盤價 = (前一根蠟燭的開盤價 + 前一根蠟燭的收盤價) / 2<br/>2. 收盤價 = (當前週期的開盤價 + 最高價 + 最低價 + 收盤價) / 4<br/>3. 最高價 = 當前週期的最高價、開盤價和收盤價中的最高值<br/>4. 最低價 = 當前週期的最低價、開盤價和收盤價中的最低值</p><p><strong>具體案例：</strong><br/>案例 1：在一個上升趨勢中，平均蠟燭圖通常會顯示連續的綠色蠟燭，且蠟燭的下影線較短或沒有下影線。這表明市場處於強勢上升階段。<br/>案例 2：在一個下降趨勢中，平均蠟燭圖通常會顯示連續的紅色蠟燭，且蠟燭的上影線較短或沒有上影線。這表明市場處於強勢下降階段。</p><p><strong>常見問題：</strong><br/>1. <strong>平均蠟燭圖是否適合短期交易？</strong> 平均蠟燭圖更適合中長期趨勢分析，因為它平滑了價格波動，可能會延遲信號。<br/>2. <strong>為什麼平均蠟燭圖看不到價格缺口？</strong> 由於其計算方法，平均蠟燭圖隱藏了價格缺口，這有助於減少市場噪音，但也可能遺漏一些重要的價格信息。</p>`} id={101562} />
