# 目標搜索

<ArticleMeta id={100784} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

目標搜索是在只知道輸出值的情況下尋找正確輸入值的過程。目標搜索功能可以嵌入不同類型的計算機軟件程序中，如 Microsoft Excel。
