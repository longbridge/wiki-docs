# 信用違約互換

<ArticleMeta id={102170} updatedAt={'2024-09-12 02:38:27'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>信用違約互換（Credit Default Swap, CDS）是一種金融衍生工具，允許一方（保護買方）向另一方（保護賣方）支付定期費用，以換取在參考實體（如公司或政府）發生信用事件（如違約）時，保護賣方對保護買方進行補償。CDS 用於對沖信用風險或進行信用投機，常見於債券市場和信用衍生品市場。</p><p>主要特點包括：</p><p>信用保護：保護買方通過支付定期保費獲得信用保護，如果參考實體發生違約，保護賣方將支付賠償。<br/>參考實體：通常為公司債券、政府債券或其他債務工具。<br/>信用事件：包括違約、破產、重組等事件，一旦發生即觸發 CDS 的賠償條款。<br/>市場交易：CDS 在場外交易市場（OTC）交易，合同條款可定製。<br/>信用違約互換的應用示例：<br/>假設一家投資公司持有價值 1000 萬美元的公司債券，為對沖該債券的違約風險，投資公司購買了一份 CDS。每年支付的保費為 50 萬美元。如果該公司債券發生違約，CDS 的保護賣方將支付 1000 萬美元的賠償給投資公司，從而使其免受損失。</p>
