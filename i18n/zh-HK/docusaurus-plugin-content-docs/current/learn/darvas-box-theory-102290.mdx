# 達爾瓦斯盒理論

<ArticleMeta id={102290} updatedAt={'2024-09-12 16:55:51'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>達爾瓦斯盒理論（Darvas Box Theory）是一種技術分析方法，由匈牙利舞蹈家和投資者尼古拉斯·達爾瓦斯（Nicolas Darvas）在 20 世紀 50 年代創立。該理論通過繪製價格區間（盒子）來識別股票的買入和賣出點。具體來説，當股票價格突破上一個盒子的上限時，達爾瓦斯認為這是一個買入信號；相反，當股票價格跌破當前盒子的下限時，則是一個賣出信號。這種方法強調跟隨趨勢和價格動量，通過設定止損點來管理風險。達爾瓦斯盒理論簡單易懂，被許多投資者用來捕捉價格突破帶來的交易機會。</p>
