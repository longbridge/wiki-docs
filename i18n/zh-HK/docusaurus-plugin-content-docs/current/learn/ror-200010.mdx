# RoR

<ArticleMeta id={200010} updatedAt={'2024-09-12 11:58:00'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p><span style={{backgroundColor:"rgb(255,255,255)",color:"rgb(17,17,17)"}}>RoR 是 Rate of Return 的縮寫，也即 “回報率”。</span></p><p><span style={{backgroundColor:"rgb(255,255,255)",color:"rgb(17,17,17)"}}>RoR 是投資在特定時期內的淨收益或損失，以投資初始成本的百分比表示。</span></p>

<AIContent content={`<p><strong>定義：</strong><br/>RoR 是 Rate of Return 的縮寫，也即 “回報率”。RoR 是投資在特定時期內的淨收益或損失，以投資初始成本的百分比表示。它是衡量投資績效的一個重要指標，幫助投資者評估其投資的盈利能力。</p><p><strong>起源：</strong><br/>回報率的概念可以追溯到早期的金融市場，當時投資者需要一種方法來衡量其投資的收益。隨着金融市場的發展，回報率逐漸成為評估投資績效的標準工具。</p><p><strong>類別與特點：</strong><br/>1. <strong>年化回報率（Annualized Rate of Return）：</strong>這是將投資的回報率標準化為每年的回報率，便於不同投資期限的比較。<br/>2. <strong>實際回報率（Real Rate of Return）：</strong>這是扣除通貨膨脹影響後的回報率，更能反映投資的實際收益。<br/>3. <strong>名義回報率（Nominal Rate of Return）：</strong>這是未扣除通貨膨脹影響的回報率，通常用於短期投資。</p><p><strong>具體案例：</strong><br/>1. <strong>股票投資：</strong>假設你在 2023 年初以 100 元購買了一隻股票，到 2023 年底該股票價格上漲到 120 元，同時你還獲得了 5 元的股息。那麼，RoR = [(120 + 5 - 100) / 100] * 100% = 25%。<br/>2. <strong>房地產投資：</strong>假設你在 2020 年以 50 萬元購買了一套房產，2024 年你以 70 萬元賣出，同時期間你每年獲得 2 萬元的租金收入。那麼，RoR = [(70 + (2*4) - 50) / 50] * 100% = 48%。</p><p><strong>常見問題：</strong><br/>1. <strong>如何計算回報率？</strong> 回報率的計算公式為：RoR = [(期末價值 - 初始投資 + 收益) / 初始投資] * 100%。<br/>2. <strong>回報率是否考慮風險？</strong> 回報率本身不考慮風險，投資者需要結合其他指標（如夏普比率）來評估風險調整後的回報。</p>`} id={200010} />
