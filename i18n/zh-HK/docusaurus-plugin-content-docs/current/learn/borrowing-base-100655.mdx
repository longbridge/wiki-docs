# 借貸基數

<ArticleMeta id={100655} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

借貸基數是指借款人願意根據企業抵押品的價值來借給企業的金額。借貸基數通常是通過一種稱為 “融資” 的方法來確定的，其中借方確定一個折扣因子，然後將其乘以有關抵押品的價值。得出的數值代表着借款人將向公司借出的金額。
