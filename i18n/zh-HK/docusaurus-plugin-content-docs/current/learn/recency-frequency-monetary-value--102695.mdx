# 最近性，頻率，金額

<ArticleMeta id={102695} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

最近性，頻率，金額（RFM）是一種在營銷分析中使用的模型，通過購買模式或習慣來劃分公司的消費者羣體。具體而言，它評估客户的最近購買時間，購買頻率以及購買金額。然後使用 RFM 來識別公司或組織的最佳客户，通過測量和分析消費習慣來改善得分低的客户並維護得分高的客户。
