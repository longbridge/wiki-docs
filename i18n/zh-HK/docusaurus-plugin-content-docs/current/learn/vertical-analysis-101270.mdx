# 縱向分析

<ArticleMeta id={101270} updatedAt={'2023-09-25 22:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>縱向分析是一種用於財務報表分析的技術，用來顯示每個賬户相對於總金額的相對大小。縱向分析將財務報表上的每一項表達為總金額的百分比。通過將各項表達為百分比而不是絕對數額，縱向分析使得比較不同銷售額或資產水平的公司變得更加容易。這種方法可以用於資產負債表、損益表和現金流量表。</p><p>進行縱向分析時，所有項目都表示為同一年度財務報表中某一基數的百分比。例如，縱向分析可能將資產、負債或權益賬户表示為總資產、總負債或總權益的百分比。</p>
