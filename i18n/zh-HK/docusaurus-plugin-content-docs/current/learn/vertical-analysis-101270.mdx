# 縱向分析

<ArticleMeta id={101270} updatedAt={'2023-09-26 13:58:54'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

縱向分析是一種財務報表分析的方法，其中每個項目以報表內的基數為百分比列出。因此，在損益表上的項目可以作為總銷售額的百分比，而資產負債表上的項目可以作為總資產或總負債的百分比，而現金流量表的縱向分析則將每個現金流入或流出作為總現金流入的百分比。
