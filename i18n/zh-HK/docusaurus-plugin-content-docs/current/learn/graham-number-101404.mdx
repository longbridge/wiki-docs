# 格雷厄姆數

<ArticleMeta id={101404} updatedAt={'2023-09-25 22:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

格雷厄姆數（或本傑明·格雷厄姆數）通過考慮公司每股收益（EPS）和每股賬面價值（BVPS）來衡量股票的基本價值。 格雷厄姆數是防守型投資者應為股票支付的價格範圍的上限。 根據這一理論，任何低於格雷厄姆數的股價都被認為是被低估的，因此值得投資。
