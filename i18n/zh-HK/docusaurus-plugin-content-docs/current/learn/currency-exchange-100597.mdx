# 貨幣兑換

<ArticleMeta id={100597} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

貨幣兑換是一種持牌業務，允許客户將一種貨幣兑換成另一種貨幣。紙幣和硬幣的貨幣兑換通常在櫃枱台頭完成，這些櫃枱可以在各種地方找到，如機場、銀行、酒店和度假村。貨幣兑換通過收取一定的費用和通過匯率差賺取利潤。也被稱為 “兑換處” 或 “兑換所”，貨幣兑換不應與外匯市場混淆，外匯市場是交易商和金融機構進行貨幣交易的地方。
