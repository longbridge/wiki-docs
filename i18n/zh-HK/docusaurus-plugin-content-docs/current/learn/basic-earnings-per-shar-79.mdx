# 基本每股收益

<ArticleMeta id={79} updatedAt={'2024-09-12 21:07:45'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

基本每股收益是指公司在一定會計期間內所創造的純利潤按每股平均分配給普通股股東的金額。它是投資者評估股票盈利能力的重要指標之一。

<AIContent content={`<p><strong>定義：</strong>基本每股收益（Earnings Per Share, EPS）是指公司在一定會計期間內所創造的純利潤按每股平均分配給普通股股東的金額。它是投資者評估股票盈利能力的重要指標之一。</p><p><strong>起源：</strong>基本每股收益的概念起源於 20 世紀初，隨着現代公司財務報表的普及，EPS 逐漸成為衡量公司盈利能力的重要指標。特別是在 20 世紀 60 年代，隨着證券市場的發展，EPS 成為投資者和分析師評估公司業績的標準工具。</p><p><strong>類別與特點：</strong>基本每股收益可以分為以下幾類：<ul><li><strong>基本 EPS：</strong>這是最常見的形式，計算公式為：<br/><code>基本 EPS = (淨利潤 - 優先股股息) / 流通在外普通股加權平均數</code></li><li><strong>稀釋 EPS：</strong>考慮了潛在的稀釋因素，如可轉換債券、期權等，計算公式為：<br/><code>稀釋 EPS = (淨利潤 - 優先股股息) / (流通在外普通股加權平均數 + 潛在稀釋普通股)</code></li></ul>基本 EPS 的特點是簡單易懂，能夠直接反映公司每股的盈利能力，而稀釋 EPS 則提供了更為保守的盈利能力評估。</p><p><strong>具體案例：</strong><ul><li><strong>案例一：</strong>某公司在 2023 年實現淨利潤 1000 萬元，流通在外普通股加權平均數為 500 萬股，則其基本 EPS 為：<br/><code>基本 EPS = 1000 萬元 / 500 萬股 = 2 元/股</code></li><li><strong>案例二：</strong>另一家公司在 2023 年實現淨利潤 2000 萬元，流通在外普通股加權平均數為 1000 萬股，同時有 100 萬股的潛在稀釋普通股，則其稀釋 EPS 為：<br/><code>稀釋 EPS = 2000 萬元 / (1000 萬股 + 100 萬股) = 1.82 元/股</code></li></ul></p><p><strong>常見問題：</strong><ul><li><strong>問：</strong>基本 EPS 和稀釋 EPS 有什麼區別？<br/><strong>答：</strong>基本 EPS 只考慮當前流通在外的普通股，而稀釋 EPS 則考慮了所有潛在的稀釋因素，如可轉換債券和期權。</li><li><strong>問：</strong>為什麼基本 EPS 對投資者很重要？<br/><strong>答：</strong>基本 EPS 是衡量公司盈利能力的重要指標，能夠幫助投資者評估公司每股的盈利情況，從而做出更明智的投資決策。</li></ul></p>`} id={79} />
