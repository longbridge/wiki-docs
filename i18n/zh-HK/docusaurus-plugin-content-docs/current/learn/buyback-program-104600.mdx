# 回購計劃

<ArticleMeta id={104600} updatedAt={'2023-10-25 20:20:24'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

回購計劃是企業回購自己的股票的計劃或政策。回購計劃通常旨在提高每股股價和股東價值，以及減少流通股數。
