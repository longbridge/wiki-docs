# 每筆交易商品數量

<ArticleMeta id={102565} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

每筆交易商品數量 (UPT) 是常用於零售銷售業的銷售指標，用於衡量每筆交易中顧客平均購買的商品數量。UPT 越高，顧客每次訪問購買的商品數量就越多。
