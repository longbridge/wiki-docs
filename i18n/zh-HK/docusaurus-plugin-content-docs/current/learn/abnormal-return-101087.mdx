# 異常收益

<ArticleMeta id={101087} updatedAt={'2023-09-26 13:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>異常收益是指投資組合或單個股票的實際收益超出或低於預期收益的部分，通常用於評估投資表現或市場反應。</p>

<AIContent content={`<p><strong>定義：</strong>異常收益是指投資組合或單個股票的實際收益超出或低於預期收益的部分。預期收益通常基於市場平均收益、風險調整後的收益或其他基準收益。異常收益常用於評估投資表現或市場反應。</p><p><strong>起源：</strong>異常收益的概念起源於金融學中的資本資產定價模型（CAPM）和有效市場假説（EMH）。CAPM 模型通過風險和預期收益的關係來預測資產的合理收益，而 EMH 則假設市場價格已經反映了所有可用信息，因此任何超出預期的收益都被視為異常。</p><p><strong>類別與特點：</strong>異常收益可以分為正異常收益和負異常收益。正異常收益表示實際收益高於預期收益，通常被視為投資成功的標誌；負異常收益則表示實際收益低於預期收益，可能表明投資決策存在問題。異常收益的特點包括：1. 難以預測：由於市場信息的複雜性和不確定性，異常收益難以提前預測。2. 短期波動：異常收益通常是短期現象，長期來看市場會趨於均衡。3. 信息敏感：市場上的新信息（如公司財報、經濟數據）會迅速反映在價格中，導致異常收益的產生。</p><p><strong>具體案例：</strong>案例一：某公司發佈了超出市場預期的季度財報，導致其股票價格在短時間內大幅上漲，投資者因此獲得了正異常收益。案例二：某國央行意外宣佈加息，市場未能提前預料到這一信息，導致股市整體下跌，投資者因此遭受了負異常收益。</p><p><strong>常見問題：</strong>1. 如何計算異常收益？通常使用實際收益減去預期收益的方法。2. 異常收益是否可持續？由於市場趨於均衡，異常收益通常是短期現象，難以長期持續。</p>`} id={101087} />
