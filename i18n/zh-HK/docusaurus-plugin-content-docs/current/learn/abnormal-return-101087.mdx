# 異常收益

<ArticleMeta id={101087} updatedAt={'2023-09-26 13:58:54'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

異常收益是指在特定期間內，給定投資或投資組合產生的異常大的利潤或損失。其表現與預期的投資回報率（RoR）有所不同，預期回報率是基於資產定價模型、長期歷史平均值或多種估值技術的風險調整回報率的估計。異常收益可能僅僅是異常的，也可能指示更邪惡的事情，比如欺詐或操縱。異常收益不應與主動管理投資所獲得的 “阿爾法” 或超額收益混淆。
