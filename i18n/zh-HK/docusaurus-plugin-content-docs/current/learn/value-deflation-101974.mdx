# 價值通貨緊縮

<ArticleMeta id={101974} updatedAt={'2024-09-12 03:54:21'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

價值通貨緊縮，又稱縮水通脹，是指零售商和服務提供商削減成本，以相同的價格銷售更小的包裝，提供更小的份量，或者總體上提供更少的產品，以保持相同的價格。當成本上漲和消費者特別注重價格時，企業可能會以這種方式隱蔽地提高價格。從經濟範圍上看，價值通貨緊縮實際上是一種價格通脹，在相同的價格水平上導致了較低的實際消費。如果在價格指數的計算中沒有考慮到價值通貨緊縮，它可能會導致通脹率和生活成本被低估。價值通貨緊縮是一種 “隱藏的通脹”，反映在定量變化上，這些變化難以用傳統通脹指數來追蹤。例如，公司可以選擇在生產線上偷工減料，生產出不那麼耐用的商品。或者他們可以添加防腐劑，延長之前作為新鮮農產品銷售的貨物的保質期。
