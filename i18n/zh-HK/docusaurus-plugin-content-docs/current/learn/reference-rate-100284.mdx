# 參考利率

<ArticleMeta id={100284} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

參考利率是用於設定其他利率的利率基準。不同類型的交易使用不同的參考利率基準，但最常見的包括聯邦基金利率、倫敦銀行同業拆借利率 (LIBOR)、基準美國國債利率和基準聯邦儲備證券利率。參考利率在房主按揭貸款和機構進行復雜利率互換交易中非常有用。
