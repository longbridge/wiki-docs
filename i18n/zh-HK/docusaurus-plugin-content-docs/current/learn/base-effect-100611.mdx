# 基準效應

<ArticleMeta id={100611} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

基準效應指的是在比較兩個數據點時，所選擇的不同參考點對於比較結果可能產生的影響。這通常涉及到在時間序列數據集中兩個點之間使用某種比率或指數值，但也可以應用於橫截面或其他類型的數據。在比較不同的數字或數據時考慮基準效應意味着考慮到一個問題：“相比於什麼？” 選擇比較的基準對於比較結果的顯著影響。如果忽視或誤解了基準效應，可能會導致嚴重失真和可能的錯誤結論；然而，如果仔細考慮，可以利用基準效應來改進分析師對數據及其生成過程的理解。
