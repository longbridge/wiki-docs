# 立即成交或取消訂單

<ArticleMeta id={102664} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

立即成交或取消訂單（IOC）是一種買入或賣出證券的訂單，試圖立即執行全部或部分，並取消未成交部分的訂單。IOC 訂單是投資者可以使用的幾種 “有效期” 或 “時間段” 的訂單之一，用於指定訂單在市場中保持有效的時間和取消訂單的條件。其他常用的有效期訂單類型包括填或殺（FOK），全部或無 ( AON) 和完全撤銷（GTC）。大多數在線交易平台允許手動或編程方式放置 IOC 訂單到自動交易策略中。
