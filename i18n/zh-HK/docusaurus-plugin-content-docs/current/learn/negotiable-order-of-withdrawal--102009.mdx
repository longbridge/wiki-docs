# 可轉讓提款令

<ArticleMeta id={102009} updatedAt={'2024-09-12 03:51:34'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

可轉讓提款令賬户是一種帶有利息的即期存款賬户。持有這種賬户的客户被允許以存款為基礎書寫匯票。可轉讓提款令賬户也被稱為 “NOW 賬户”。

<AIContent content={`<p><strong>定義：</strong>可轉讓提款令賬户（Negotiable Order of Withdrawal Account，簡稱 NOW 賬户）是一種帶有利息的即期存款賬户。持有這種賬户的客户可以在賬户餘額的基礎上書寫匯票或支票。</p><p><strong>起源：</strong>NOW 賬户最早出現在 20 世紀 70 年代的美國。當時，銀行為了吸引更多的存款客户，推出了這種既能賺取利息又能方便提款的賬户類型。1974 年，美國國會通過了《銀行法修正案》，正式允許銀行提供 NOW 賬户。</p><p><strong>類別與特點：</strong>NOW 賬户主要有以下幾個特點：<ul><li>利息收益：與普通支票賬户不同，NOW 賬户會根據賬户餘額支付利息。</li><li>流動性：持有者可以隨時提款或書寫支票，享有較高的流動性。</li><li>最低餘額要求：大多數 NOW 賬户要求持有者保持一定的最低餘額，否則可能會收取服務費。</li></ul></p><p><strong>具體案例：</strong><ol><li>案例一：小明在某銀行開設了一個 NOW 賬户，每月存入工資並支付日常開銷。由於賬户餘額較高，小明每月還能獲得一定的利息收入。</li><li>案例二：某小企業主在銀行開設了 NOW 賬户，用於日常經營的資金週轉。通過該賬户，企業主不僅能方便地支付供應商款項，還能獲得利息收益。</li></ol></p><p><strong>常見問題：</strong><ul><li>問：NOW 賬户與普通支票賬户有何區別？<br/>答：NOW 賬户會支付利息，而普通支票賬户通常不支付利息。</li><li>問：開設 NOW 賬户需要注意什麼？<br/>答：需要注意賬户的最低餘額要求和可能的服務費。</li></ul></p>`} id={102009} />
