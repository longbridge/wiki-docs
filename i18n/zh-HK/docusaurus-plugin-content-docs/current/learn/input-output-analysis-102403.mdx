# 輸入 - 輸出分析

<ArticleMeta id={102403} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

輸入 - 輸出分析（I-O）是一種基於不同經濟部門或產業之間相互依賴關係的宏觀經濟分析方法。這種方法常用於估計正面或負面經濟衝擊的影響，並分析對整個經濟的漣漪效應。輸入 - 輸出經濟分析最早由瓦西里·勒昂季耶夫（1906-1999）開發，他後來因在這一領域的工作而獲得諾貝爾經濟學獎。
