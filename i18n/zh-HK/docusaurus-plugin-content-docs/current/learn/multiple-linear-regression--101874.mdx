# 多元線性迴歸

<ArticleMeta id={101874} updatedAt={'2023-09-26 13:58:58'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

多元線性迴歸（MLR），也簡稱為多元迴歸，是一種使用多個解釋變量來預測響應變量結果的統計技術。多元線性迴歸的目標是建立解釋變量和響應變量之間的線性關係模型。本質上，多元迴歸是最小二乘（OLS）迴歸的延伸，因為它涉及多個解釋變量。
