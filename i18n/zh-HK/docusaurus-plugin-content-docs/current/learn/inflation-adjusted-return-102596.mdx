# 通脹調整後回報率

<ArticleMeta id={102596} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

通脹調整後的回報率是考慮到時期通脹率的回報率衡量指標。通脹調整後回報率的目的是顯示在消除通脹影響後的投資回報率。從投資回報率中消除通脹的影響使投資者能夠看到證券的真正盈利潛力，而不受外部經濟力量的影響。通脹調整後的回報率也稱為實際回報率或通脹調整後的所需回報率。
