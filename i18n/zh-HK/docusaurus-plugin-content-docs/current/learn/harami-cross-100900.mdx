# 十字孕育

十字孕育是一種日本蠟燭圖形態，由一個沿趨勢方向移動的大蜡燭形態，後面跟着一個小的十字蠟燭形態。該十字蠟燭形態完全包含在前一個蠟燭的實體內。十字孕育形態表明前一趨勢可能即將逆轉。該形態可能為看漲或看跌。看漲形態表示可能價格逆轉向上，而看跌形態則表示可能價格逆轉向下。
