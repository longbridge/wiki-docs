# A 類普通股

<ArticleMeta id={103209} updatedAt={'2023-10-03 08:20:17'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

A 類普通股是一種股票類別，通常具有更高的投票權和更高的優先權。A 類股票通常不向公眾發行，而是由公司的創始人、高管和董事會成員持有。這種股票通常與 B 類普通股等其他類別的股票一起發行。
