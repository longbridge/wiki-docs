# 目標累計贖回票據

<ArticleMeta id={102568} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

目標累計贖回票據（TARN）是一種異類衍生品，當對持有人的優惠支付達到限額時終止。目標累計贖回票據（TARN）有一個獨特的特徵，即能夠提前終止。如果在結算日之前優惠支付的累計金額達到預定金額，票據持有人將獲得面值的最後一次支付，合同終止。
