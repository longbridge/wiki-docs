# 訂單審核跟蹤系統

<ArticleMeta id={102508} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

訂單審核跟蹤系統（OATS）是由金融業監管機構金融業監管局（FINRA）建立的自動化計算機系統。它用於記錄與全國市場系統（NMS）上交易的所有權益，包括場外交易（OTC）股票的訂單、報價和其他相關交易數據。該系統簡化了訂單從初始接收到最終執行或取消的過程，以便於追蹤或審計目的。
