# 固定收益證券

<ArticleMeta id={101935} updatedAt={'2023-09-26 13:58:58'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

固定收益證券是一種通過固定週期的利息支付和到期時的本金歸還提供回報的投資。與可變收益證券不同，固定收益證券的回報是已知的，不會根據基礎指標（如短期利率）的變化而變動。
