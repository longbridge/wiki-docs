# 權益成本

<ArticleMeta id={100855} updatedAt={'2023-09-26 13:58:51'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

權益成本是公司為確定一個投資是否符合資本回報要求所需的回報。公司通常將其作為投資回報的資本預算閾值。公司的權益成本代表了市場對擁有這一資產並承擔所有權風險所要求的補償。權益成本的傳統公式為股利資本化模型和資本資產定價模型（CAPM）。

<AIContent content={`<p><strong>定義：</strong>權益成本是公司為確定一個投資是否符合資本回報要求所需的回報。公司通常將其作為投資回報的資本預算閾值。公司的權益成本代表了市場對擁有這一資產並承擔所有權風險所要求的補償。</p><p><strong>起源：</strong>權益成本的概念起源於 20 世紀中期，隨着現代金融理論的發展而逐漸成型。特別是威廉·夏普在 1964 年提出的資本資產定價模型（CAPM），為計算權益成本提供了理論基礎。</p><p><strong>類別與特點：</strong>權益成本主要通過兩種方法計算：股利資本化模型和資本資產定價模型（CAPM）。<br/>1. <strong>股利資本化模型：</strong>該模型基於公司未來股利的現值來計算權益成本，公式為：<br/><code>權益成本 = (下一年預期股利 / 當前股價) + 股利增長率</code>。<br/>2. <strong>資本資產定價模型（CAPM）：</strong>該模型考慮了系統性風險，公式為：<br/><code>權益成本 = 無風險利率 + β * (市場回報率 - 無風險利率)</code>。</p><p><strong>具體案例：</strong><br/>1. <strong>案例一：</strong>假設某公司當前股價為 100 元，預計下一年股利為 5 元，股利增長率為 4%。根據股利資本化模型，權益成本為：<br/><code>權益成本 = (5 / 100) + 0.04 = 0.09 或 9%</code>。<br/>2. <strong>案例二：</strong>假設某公司β值為 1.2，無風險利率為 3%，市場回報率為 8%。根據 CAPM 模型，權益成本為：<br/><code>權益成本 = 0.03 + 1.2 * (0.08 - 0.03) = 0.09 或 9%</code>。</p><p><strong>常見問題：</strong><br/>1. <strong>如何選擇合適的模型計算權益成本？</strong>選擇模型取決於公司股利政策和市場數據的可獲得性。如果公司定期支付股利且股利增長穩定，股利資本化模型較為適用；否則，CAPM 模型更為常用。<br/>2. <strong>權益成本是否固定不變？</strong>權益成本會隨着市場條件、公司風險狀況和無風險利率的變化而變化。</p>`} id={100855} />
