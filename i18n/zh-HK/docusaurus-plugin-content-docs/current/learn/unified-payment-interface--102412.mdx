# 統一支付接口

<ArticleMeta id={102412} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

統一支付接口 (UPI) 是一款智能手機應用程序，允許用户在銀行賬户之間進行轉賬。它是由印度國家支付公司 (NPCI) 開發的一種單一窗口移動支付系統。每當客户發起交易時，它消除了輸入銀行詳細信息或其他敏感信息的需要。
