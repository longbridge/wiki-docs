# 平均資本回報率

<ArticleMeta id={102379} updatedAt={'2024-09-12 16:33:53'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

平均資本回報率 (ROAE) 是一種衡量公司績效的財務比率，基於平均流通股東權益。通常，ROAE 指的是一家公司在一個財政年度內的績效，所以 ROAE 的分子是淨收入，分母是計算年初和年末的權益價值之和除以 2。平均資本回報率不同於更常見的股東權益回報率 (ROE)，後者衡量的是年度淨收入除以年末股東權益的金額，該金額可能受股票銷售、股息支付和其他股份稀釋的影響。

<AIContent content={`<p><strong>定義：</strong>平均資本回報率（Return on Average Equity，簡稱 ROAE）是一種衡量公司績效的財務比率，基於平均流通股東權益。ROAE 的計算公式為：<br/>ROAE = 淨收入 / 平均股東權益<br/>其中，平均股東權益是年初和年末股東權益的平均值。</p><p><strong>起源：</strong>ROAE 的概念源於對公司財務績效的深入分析需求。隨着財務分析方法的不斷發展，投資者和分析師需要一種更準確的方式來評估公司在整個財政年度內的績效表現，而不僅僅是基於年末的股東權益。</p><p><strong>類別與特點：</strong>ROAE 主要用於評估公司在一個財政年度內的整體績效。與更常見的股東權益回報率（ROE）相比，ROAE 通過使用平均股東權益來平滑年度內的波動，提供了一個更穩定的績效指標。<br/>1. <strong>優點：</strong>ROAE 能夠更準確地反映公司在整個年度內的績效，減少了因股票銷售、股息支付和其他股份稀釋對年末股東權益的影響。<br/>2. <strong>缺點：</strong>由於需要計算平均股東權益，ROAE 的計算相對複雜，且可能不適用於所有公司，特別是那些股東權益波動較大的公司。</p><p><strong>具體案例：</strong><br/>1. <strong>案例一：</strong>某公司在 2023 年初的股東權益為 1000 萬元，年末為 1200 萬元，淨收入為 200 萬元。其 ROAE 計算如下：<br/>平均股東權益 = (1000 + 1200) / 2 = 1100 萬元<br/>ROAE = 200 / 1100 = 18.18%<br/>這表明該公司在 2023 年的平均資本回報率為 18.18%。<br/>2. <strong>案例二：</strong>另一家公司在 2023 年初的股東權益為 500 萬元，年末為 700 萬元，淨收入為 100 萬元。其 ROAE 計算如下：<br/>平均股東權益 = (500 + 700) / 2 = 600 萬元<br/>ROAE = 100 / 600 = 16.67%<br/>這表明該公司在 2023 年的平均資本回報率為 16.67%。</p><p><strong>常見問題：</strong><br/>1. <strong>ROAE 和 ROE 有什麼區別？</strong><br/>ROAE 使用平均股東權益，而 ROE 使用年末股東權益。ROAE 更能反映公司在整個年度內的績效，而 ROE 可能受年末股東權益波動的影響。<br/>2. <strong>為什麼要使用 ROAE 而不是 ROE？</strong><br/>ROAE 通過平滑年度內的波動，提供了一個更穩定的績效指標，特別適用於股東權益波動較大的公司。</p>`} id={102379} />
