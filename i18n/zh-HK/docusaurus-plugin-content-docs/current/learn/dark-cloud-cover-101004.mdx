# 烏雲壓頂

<ArticleMeta id={101004} updatedAt={'2023-09-26 13:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

烏雲壓頂是一種看跌反轉的蠟燭圖形式，由向上和向下的兩根蠟燭線組成。首先是一根向上的蠟燭線，然後是一根向下的蠟燭線。烏雲壓頂的特點是第二根蠟燭線的開盤價高於前一根蠟燭線的收盤價，然後收盤價在前一根蠟燭線的中點以下。這個圖形顯示了市場的動力從上漲轉向下跌。交易者尋求價格在下一根蠟燭線上進一步下降作為確認信號。
