# 絕對回報

<ArticleMeta id={100672} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

絕對回報是指在指定期間內資產實現的回報。這個指標以百分比表示了資產（如股票或共同基金）在給定期間內的升值或貶值。絕對回報與相對回報的不同之處在於，它關注的是特定資產的回報，並不將其與其他指標或基準進行比較。
