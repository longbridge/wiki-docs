# 平均年回報率

<ArticleMeta id={103025} updatedAt={'2023-09-29 02:20:38'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

平均年化收益率是指資產或投資組合在一段時間內的平均年化收益率。它是一種重要的投資指標，用於評估資產或投資組合的長期表現。平均年化收益率可以根據不同的時間段來計算，通常以年為單位。
