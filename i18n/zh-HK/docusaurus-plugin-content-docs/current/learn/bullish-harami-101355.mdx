# 看漲孕線

<p>看漲孕線是一種技術分析中的 K 線形態，常用於預測市場的反轉趨勢。該形態由兩根 K 線組成：第一根是較長的陰線，表示市場的下跌趨勢；第二根是較短的陽線，完全包含在第一根陰線的實體之內。這個形態表明市場的賣壓可能正在減弱，買方力量開始增強。看漲孕線的出現通常被視為潛在的買入信號，尤其是在下跌趨勢的末期。這一形態的有效性可以通過結合其他技術指標（如交易量、相對強弱指數等）來確認。</p>
