# 按揭支持證券

<ArticleMeta id={101789} updatedAt={'2024-09-12 04:04:40'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

按揭支持證券（MBS）是類似債券的投資產品。每個 MBS 由一束從發行它們的銀行購買的住房貸款和其他房地產債務組成。投資按揭支持證券的投資者會收到類似債券利息支付的週期性支付。

<AIContent content={`<p><strong>定義：</strong>按揭支持證券（Mortgage-Backed Securities，簡稱 MBS）是一種類似債券的投資產品。每個 MBS 由一束從發行它們的銀行購買的住房貸款和其他房地產債務組成。投資按揭支持證券的投資者會收到類似債券利息支付的週期性支付。</p><p><strong>起源：</strong>按揭支持證券的概念起源於 20 世紀 70 年代的美國。當時，美國政府為了促進住房市場的發展，推出了這種金融工具。1970 年，政府國民抵押協會（Ginnie Mae）發行了第一批 MBS，隨後聯邦國民抵押協會（Fannie Mae）和聯邦住房貸款抵押公司（Freddie Mac）也開始發行 MBS。</p><p><strong>類別與特點：</strong>按揭支持證券主要分為兩類：機構 MBS 和非機構 MBS。<ul><li><strong>機構 MBS：</strong>由政府支持的機構（如 Ginnie Mae、Fannie Mae 和 Freddie Mac）發行，具有較高的信用評級和較低的風險。</li><li><strong>非機構 MBS：</strong>由私人金融機構發行，風險較高，但也可能提供更高的收益。</li></ul>按揭支持證券的特點包括：<ul><li>週期性支付：投資者會收到類似債券利息支付的週期性支付。</li><li>風險分散：由於 MBS 由多筆貸款組成，單個貸款違約的風險被分散。</li><li>流動性：MBS 在二級市場上交易，具有一定的流動性。</li></ul></p><p><strong>具體案例：</strong><ul><li><strong>案例一：</strong>某投資者購買了一批由 Fannie Mae 發行的 MBS。這些 MBS 由一組住房貸款組成，每月投資者會收到來自這些貸款的利息支付。由於 Fannie Mae 的信用評級較高，這些 MBS 的風險較低。</li><li><strong>案例二：</strong>某投資者購買了一批由私人金融機構發行的非機構 MBS。這些 MBS 由一組商業房地產貸款組成，雖然這些 MBS 的收益較高，但由於發行機構的信用評級較低，投資者需要承擔更高的風險。</li></ul></p><p><strong>常見問題：</strong><ul><li><strong>問：</strong>按揭支持證券的主要風險是什麼？<br/><strong>答：</strong>主要風險包括利率風險、信用風險和提前償還風險。</li><li><strong>問：</strong>如何選擇合適的按揭支持證券？<br/><strong>答：</strong>投資者應根據自身的風險承受能力、收益預期和市場狀況選擇合適的 MBS。</li></ul></p>`} id={101789} />
