# 依賴比率

<ArticleMeta id={101086} updatedAt={'2024-09-13 10:59:44'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>依賴比率是指年齡在 0 至 14 歲以及 65 歲以上的被撫養人口與年齡在 15 至 64 歲的總人口的比率。這個人口統計指標揭示了非勞動年齡人口的數量與勞動年齡人口的數量之間的關係。它還用於瞭解勞動力的相對經濟負擔，並且對税收產生影響。</p>
