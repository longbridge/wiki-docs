# 真實強度指數

<ArticleMeta id={102461} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

真實強度指數 (TSI) 是一個技術動量震盪器，用於識別趨勢和反轉。該指標可能有助於確定超買和超賣條件，通過中線或信號線的交叉指示可能的趨勢方向變化，並通過背離警示趨勢的弱勢。
