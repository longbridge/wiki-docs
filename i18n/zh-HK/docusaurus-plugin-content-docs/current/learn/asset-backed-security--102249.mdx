# 資產支持證券

<ArticleMeta id={102249} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

資產支持證券 (ABS) 是一種以擔保資產為抵押的金融投資工具，通常是以債務形式產生現金流的資產作為擔保，如貸款、租賃、信用卡餘額或應收賬款。它採取債券或票據的形式，在一定的時間內以固定利率支付收入，直至到期。對於以收入為導向的投資者來説，資產支持證券可以成為其他債務工具（如公司債券或債券基金）的替代品。
