# 希斯 - 賈羅 - 莫頓模型

希斯 - 賈羅 - 莫頓模型（HJM 模型）用於模擬遠期利率。然後將這些利率建模到現有的利率期限結構中，以確定與利率敏感的證券相關的適當價格。
