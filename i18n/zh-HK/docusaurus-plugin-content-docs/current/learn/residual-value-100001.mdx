# 殘值

<ArticleMeta id={100001} updatedAt={'2023-09-26 13:58:44'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

殘值，也稱拯救價值，是指在租賃期限或使用壽命結束時估計的固定資產價值。在租賃情況下，出租人使用殘值作為確定承租人支付週期性租金的主要方法之一。一般來説，資產的使用壽命或租賃期越長，其殘值越低。
