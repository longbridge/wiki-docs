# 美國存託憑證

<ArticleMeta id={219} updatedAt={'2024-09-12 21:00:58'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>美國存託憑證（American Depositary Receipts，簡稱 ADR）是一種在美國證券市場上交易的外國公司股票的證券形式。ADR 可用於投資者在美國交易外國公司股票，同時也使得外國公司能夠更便捷地在美國市場上融資和擴大知名度。</p>

<AIContent content={`<p><strong>定義：</strong>美國存託憑證（American Depositary Receipts，簡稱 ADR）是一種在美國證券市場上交易的外國公司股票的證券形式。ADR 可用於投資者在美國交易外國公司股票，同時也使得外國公司能夠更便捷地在美國市場上融資和擴大知名度。</p><p><strong>起源：</strong>ADR 的概念最早出現在 1927 年，由摩根大通銀行（J.P. Morgan）首次推出。最初的目的是為了幫助美國投資者更方便地投資英國公司 Selfridges 的股票。隨着全球化的發展，ADR 逐漸成為外國公司進入美國資本市場的重要工具。</p><p><strong>類別與特點：</strong>ADR 主要分為三類：一級 ADR、二級 ADR 和三級 ADR。<ul><li><strong>一級 ADR：</strong>不在美國主要證券交易所上市，只能在場外交易市場（OTC）交易，信息披露要求較低。</li><li><strong>二級 ADR：</strong>在美國主要證券交易所（如紐約證券交易所或納斯達克）上市，需符合較高的信息披露要求。</li><li><strong>三級 ADR：</strong>不僅在美國主要證券交易所上市，還可以通過發行新股在美國市場上融資，信息披露要求最高。</li></ul></p><p><strong>具體案例：</strong><ul><li><strong>案例一：</strong>阿里巴巴集團（Alibaba Group）在 2014 年通過發行 ADR 在紐約證券交易所上市，成為當時全球最大的 IPO 之一。這使得美國投資者能夠方便地投資這家中國科技巨頭。</li><li><strong>案例二：</strong>瑞士製藥公司諾華（Novartis）通過發行 ADR 在美國市場上交易，使得其能夠更好地接觸美國投資者，並在美國市場上籌集資金。</li></ul></p><p><strong>常見問題：</strong><ul><li><strong>問題一：</strong>投資 ADR 是否需要支付額外費用？<br/><strong>解答：</strong>是的，投資 ADR 可能需要支付存託銀行的管理費用，這些費用通常會從股息中扣除。</li><li><strong>問題二：</strong>ADR 的股息是否會受到雙重徵税？<br/><strong>解答：</strong>可能會。股息首先在公司所在國徵税，然後在美國再次徵税。不過，美國和許多國家之間有税收協定，可以減輕雙重徵税的負擔。</li></ul></p>`} id={219} />
