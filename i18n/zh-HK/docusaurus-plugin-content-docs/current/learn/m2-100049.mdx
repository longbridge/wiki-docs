# M2

<ArticleMeta id={100049} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

M2 是指某一個時間內，貨幣資產的總量，又被稱為廣義貨幣供應量。它是金融系統對貨幣供應總量的估計，一般包括所有人持有的現金以及存款賬户、儲蓄賬户和其他短期儲蓄工具（如定期存單）。 
在貨幣供應量中，還包括了 M0 和 M1。
M0 = 流動中的現金；
M1 = M0 + 商業銀行活期存款，又被稱為狹義貨幣。
而 M2 = M1 + 商業銀行定期存款。
