# 公允價值

<ArticleMeta id={100476} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

公允價值是指在買賣雙方自由同意價格情況下，資產被購買或出售的估計價格。為確定產品或金融投資的公允價值，個人或企業可參考類似資產的實際市場交易情況，估計資產預期收益，並確定資產替代成本。
