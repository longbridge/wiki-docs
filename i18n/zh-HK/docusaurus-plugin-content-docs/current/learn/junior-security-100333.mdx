# 次級證券

<ArticleMeta id={100333} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

次級證券是指優先級低於其他證券的證券。簡單來説，次級證券在任何其他類型的證券之後。這意味着如果公司破產或清算，只有持有次級證券的持有人才能得到償付。因此，很有可能公司欠債的一些 (甚至全部) 次級證券在剩下的現金分配後可能無法償還。
