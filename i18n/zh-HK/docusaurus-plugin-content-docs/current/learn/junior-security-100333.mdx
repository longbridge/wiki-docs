# 次級證券

<ArticleMeta id={100333} updatedAt={'2023-09-26 13:58:47'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>次級證券是指在資本結構中優先級較低的證券。這類證券在公司清算或破產時的償還順序排在優先證券之後。次級證券通常包括次級債券和次級股權等。由於其優先級較低，次級證券的風險較高，但相應的，它們也可能提供更高的收益率。投資者在購買次級證券時需要考慮其較高的信用風險和流動性風險。</p>
