# M0

M0 在貨幣供應量中，是指流通中貨幣，包括鑄幣在內的所有實物貨幣的總和。

<AIContent content={`<h2>什麼是 M0</h2><p>M0 在貨幣供應量中，是指流通中貨幣，包括鑄幣在內的所有實物貨幣的總和。它是貨幣供應量的最基礎層級，通常被稱為 “基礎貨幣” 或 “高能貨幣”。M0 主要包括紙幣和硬幣，這些貨幣是公眾和銀行持有的現金。</p><h3>起源</h3><p>M0 的概念起源於貨幣經濟學的發展過程中。隨着現代銀行體系的建立和貨幣政策的演變，經濟學家和政策制定者需要一種方式來衡量和控制貨幣供應量。M0 作為最基礎的貨幣供應量指標，幫助他們瞭解流通中的實際現金量。</p><h3>類別與特點</h3><p>M0 主要包括以下幾類：</p><ul><li><strong>紙幣：</strong>由中央銀行發行的法定貨幣，通常是國家的主要流通貨幣。</li><li><strong>硬幣：</strong>由政府鑄造的金屬貨幣，通常用於小額交易。</li></ul><p>特點：</p><ul><li><strong>流動性高：</strong>M0 是流動性最高的貨幣形式，可以立即用於交易。</li><li><strong>無利息：</strong>持有 M0 不會產生利息收益。</li><li><strong>易於計量：</strong>由於 M0 主要是實物貨幣，計量相對簡單。</li></ul><h3>具體案例</h3><p><strong>案例一：</strong>在日常生活中，人們使用紙幣和硬幣進行購物、支付交通費用等，這些交易都涉及 M0。比如，一個人用現金購買了一杯咖啡，這筆交易中的現金就是 M0 的一部分。</p><p><strong>案例二：</strong>在銀行體系中，銀行持有的現金儲備也是 M0 的一部分。當銀行從中央銀行提取現金以滿足客户的取款需求時，這些現金也屬於 M0。</p><h3>常見問題</h3><p><strong>問題一：</strong>為什麼 M0 不包括銀行存款？<br/><strong>解答：</strong>M0 僅包括流通中的實物貨幣，而銀行存款屬於 M1 或更高層級的貨幣供應量。</p><p><strong>問題二：</strong>持有大量 M0 有什麼風險？<br/><strong>解答：</strong>持有大量現金可能面臨被盜或丟失的風險，而且現金不會產生利息收益。</p>`} id={100047} />
