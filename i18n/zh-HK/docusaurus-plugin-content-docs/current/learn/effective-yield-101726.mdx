# 有效收益率

<ArticleMeta id={101726} updatedAt={'2023-09-25 22:58:58'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

有效收益率是指債券的回報，該債券的利息支付（或優惠券）由債券持有人以同樣的利率再投資。有效收益率是投資者獲得的總收益，與名義收益率相對應——名義收益率是債券票面利率的陳述。有效收益率考慮了複利對投資回報的影響，而名義收益率則不考慮。

<AIContent content={`<p><strong>定義：</strong>有效收益率（Effective Yield）是指債券的回報率，假設債券的利息支付（或優惠券）由債券持有人以同樣的利率再投資。有效收益率考慮了複利對投資回報的影響，而名義收益率（Nominal Yield）則不考慮複利。</p><p><strong>起源：</strong>有效收益率的概念起源於對債券投資回報的更精確計算。隨着金融市場的發展，投資者需要一種方法來衡量債券在考慮複利效應後的實際回報，因此有效收益率應運而生。</p><p><strong>類別與特點：</strong>有效收益率主要有以下特點：<ul><li>考慮複利：有效收益率計算時考慮了利息再投資的複利效應。</li><li>更精確：相比名義收益率，有效收益率提供了更精確的投資回報率。</li><li>適用範圍廣：適用於各種定期支付利息的債券。</li></ul></p><p><strong>具體案例：</strong><ol><li>假設某投資者購買了一張面值為 1000 元的債券，年利率為 5%，每半年支付一次利息。名義收益率為 5%，但如果考慮到每半年支付的利息再投資，實際的有效收益率會高於 5%。</li><li>另一例子是某投資者購買了一張年利率為 6% 的債券，每季度支付一次利息。名義收益率為 6%，但由於每季度支付的利息可以再投資，實際的有效收益率會高於 6%。</li></ol></p><p><strong>常見問題：</strong><ul><li><strong>問：</strong>有效收益率和名義收益率有什麼區別？<br/><strong>答：</strong>有效收益率考慮了利息再投資的複利效應，而名義收益率則不考慮。</li><li><strong>問：</strong>如何計算有效收益率？<br/><strong>答：</strong>有效收益率的計算公式為：<br/>有效收益率 = (1 + (名義收益率 / n))^n - 1，其中 n 為每年的複利次數。</li></ul></p>`} id={101726} />
