# 定性分析

<ArticleMeta id={100386} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

在商業和管理領域，定性分析使用主觀判斷來分析公司的價值或前景，基於不可量化的信息，如管理專長、行業循環、研發實力和勞動關係。定性分析與定量分析形成對比，定量分析側重於報告中的數字，如資產負債表。然而，這兩種技術通常會結合使用，以審查公司的運營情況並評估其作為投資機會的潛力。
