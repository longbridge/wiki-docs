# 賬面價值

<ArticleMeta id={101186} updatedAt={'2023-09-26 13:58:54'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

對於價值投資者來説，賬面價值是指公司資產負債表股東權益部分所有項目金額的總和。也可以通過從企業總資產中減去總負債來計算賬面價值。會計師還使用一種用於評估公司擁有的資產的賬面價值。這與投資者的賬面價值不同，因為它在內部用於管理會計目的。
