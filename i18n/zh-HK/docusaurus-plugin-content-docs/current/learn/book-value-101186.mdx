# 賬面價值

<ArticleMeta id={101186} updatedAt={'2023-09-25 22:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

對於價值投資者來説，賬面價值是指公司資產負債表股東權益部分所有項目金額的總和。也可以通過從企業總資產中減去總負債來計算賬面價值。會計師還使用一種用於評估公司擁有的資產的賬面價值。這與投資者的賬面價值不同，因為它在內部用於管理會計目的。

<AIContent content={`<p><strong>定義：</strong>對於價值投資者來説，賬面價值是指公司資產負債表股東權益部分所有項目金額的總和。也可以通過從企業總資產中減去總負債來計算賬面價值。會計師還使用一種用於評估公司擁有的資產的賬面價值。這與投資者的賬面價值不同，因為它在內部用於管理會計目的。</p><p><strong>起源：</strong>賬面價值的概念起源於會計學，最早可以追溯到雙重記賬法的發明。隨着現代企業制度的發展，賬面價值逐漸成為衡量公司財務健康狀況的重要指標之一。</p><p><strong>類別與特點：</strong>賬面價值可以分為兩類：1. 投資者賬面價值：用於評估公司整體價值，計算方法為總資產減去總負債。2. 會計賬面價值：用於內部管理，評估公司具體資產的價值。投資者賬面價值的特點是簡單直觀，適合快速評估公司財務狀況；會計賬面價值則更為詳細，適合內部管理和決策。</p><p><strong>具體案例：</strong>案例 1：某公司總資產為 1000 萬元，總負債為 400 萬元，則其賬面價值為 600 萬元。案例 2：某公司在資產負債表中顯示股東權益總額為 800 萬元，這也是其賬面價值。</p><p><strong>常見問題：</strong>1. 賬面價值是否等同於市場價值？不等同，市場價值是股票在市場上的交易價格，可能高於或低於賬面價值。2. 賬面價值是否能完全反映公司價值？不能，賬面價值僅反映公司財務狀況的一部分，其他因素如市場前景、品牌價值等也需考慮。</p>`} id={101186} />
