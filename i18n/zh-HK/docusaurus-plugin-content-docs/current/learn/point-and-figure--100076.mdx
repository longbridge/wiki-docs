# 點數圖

<ArticleMeta id={100076} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

點數圖 (P&F 圖) 是繪製股票、債券、商品或期貨的價格走勢圖，而不考慮時間的流逝。與其他類型的圖表（如蠟燭圖）不同，蠟燭圖標記資產在確定的時間段內的運動程度，P&F 圖利用由堆疊的 X 或 O 組成的列，每個 X 或 O 代表一定金額的價格運動。X 表示價格上漲，O 表示價格下跌。當查看 P&F 圖時，技術分析師仍然使用支撐和阻力等概念，以及其他模式。一些人認為，支撐和阻力水平以及突破在 P&F 圖上更清晰定義，因為它過濾了微小的價格變動，對虛假的突破更不敏感。
