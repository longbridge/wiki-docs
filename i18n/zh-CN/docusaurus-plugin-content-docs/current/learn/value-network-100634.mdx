# 价值网络

<ArticleMeta id={100634} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

价值网络是一组相互作用的组织和/或个人之间的连接，以使整个群体受益。价值网络允许成员购买和销售产品，并共享信息。可以使用简单的映射工具展示节点（成员）和连接器（关系）来可视化这些网络。
