# 总资产收益率

<ArticleMeta id={102261} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

总资产收益率（ROTA）是一种衡量公司营业利润与总净资产之间关系的比率。它被定义为净收入与总平均资产之间的比率，或者公司在一个财政年度内获得的财务和经营收入与该公司总资产的平均值之间的比率。该比率被认为是衡量一家公司利用其资产创造收益的有效性的指标。相比于类似公司，使用 EBIT 而不是净利润来保持该指标集中在经营收益上，而不受税收或融资差异的影响。
