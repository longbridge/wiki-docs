# 向上跳空孤峰下跌

向上跳空孤峰下跌形态是一个三天的蜡烛图形态，表明向上的价格动力可能已经消耗殆尽并可能反转下行。由于该形态涉及到三根特定的蜡烛柱按照一定的顺序排列，所以这种形态并不常见。
