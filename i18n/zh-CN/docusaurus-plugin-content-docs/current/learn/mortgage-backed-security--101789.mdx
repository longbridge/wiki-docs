# 按揭支持证券

<ArticleMeta id={101789} updatedAt={'2024-09-12 19:04:40'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

按揭支持证券（MBS）是类似债券的投资产品。每个 MBS 由一束从发行它们的银行购买的住房贷款和其他房地产债务组成。投资按揭支持证券的投资者会收到类似债券利息支付的周期性支付。

<AIContent content={`<p><strong>定义：</strong>按揭支持证券（Mortgage-Backed Securities，简称 MBS）是一种类似债券的投资产品。每个 MBS 由一束从发行它们的银行购买的住房贷款和其他房地产债务组成。投资按揭支持证券的投资者会收到类似债券利息支付的周期性支付。</p><p><strong>起源：</strong>按揭支持证券的概念起源于 20 世纪 70 年代的美国。当时，美国政府为了促进住房市场的发展，推出了这种金融工具。1970 年，政府国民抵押协会（Ginnie Mae）发行了第一批 MBS，随后联邦国民抵押协会（Fannie Mae）和联邦住房贷款抵押公司（Freddie Mac）也开始发行 MBS。</p><p><strong>类别与特点：</strong>按揭支持证券主要分为两类：机构 MBS 和非机构 MBS。<ul><li><strong>机构 MBS：</strong>由政府支持的机构（如 Ginnie Mae、Fannie Mae 和 Freddie Mac）发行，具有较高的信用评级和较低的风险。</li><li><strong>非机构 MBS：</strong>由私人金融机构发行，风险较高，但也可能提供更高的收益。</li></ul>按揭支持证券的特点包括：<ul><li>周期性支付：投资者会收到类似债券利息支付的周期性支付。</li><li>风险分散：由于 MBS 由多笔贷款组成，单个贷款违约的风险被分散。</li><li>流动性：MBS 在二级市场上交易，具有一定的流动性。</li></ul></p><p><strong>具体案例：</strong><ul><li><strong>案例一：</strong>某投资者购买了一批由 Fannie Mae 发行的 MBS。这些 MBS 由一组住房贷款组成，每月投资者会收到来自这些贷款的利息支付。由于 Fannie Mae 的信用评级较高，这些 MBS 的风险较低。</li><li><strong>案例二：</strong>某投资者购买了一批由私人金融机构发行的非机构 MBS。这些 MBS 由一组商业房地产贷款组成，虽然这些 MBS 的收益较高，但由于发行机构的信用评级较低，投资者需要承担更高的风险。</li></ul></p><p><strong>常见问题：</strong><ul><li><strong>问：</strong>按揭支持证券的主要风险是什么？<br/><strong>答：</strong>主要风险包括利率风险、信用风险和提前偿还风险。</li><li><strong>问：</strong>如何选择合适的按揭支持证券？<br/><strong>答：</strong>投资者应根据自身的风险承受能力、收益预期和市场状况选择合适的 MBS。</li></ul></p>`} id={101789} />
