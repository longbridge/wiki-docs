# 总收益指数

<ArticleMeta id={101775} updatedAt={'2024-09-12 19:05:18'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

总收益指数是一种股票指数，跟踪指数组成部分的资本收益和任何现金分配 (如股息或利息)。通过假定股息被再投资，它有效地计算了指数中那些不发放股息而是将盈利再投资到基础公司内的股票。总回报指数可以与价格回报或名义指数进行对比。
