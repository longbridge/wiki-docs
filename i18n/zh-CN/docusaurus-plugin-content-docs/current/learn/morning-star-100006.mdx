# 晨星

晨星是由三根蜡烛线形成的视觉图案，被技术分析师解读为看涨信号。晨星形成在下降趋势之后，表明上升趋势开始。它是先前价格趋势逆转的迹象。交易者会观察晨星的形成，并使用其他指标来确认逆转是否真正发生。
