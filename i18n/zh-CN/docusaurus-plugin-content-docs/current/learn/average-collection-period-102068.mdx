# 平均收款周期

<ArticleMeta id={102068} updatedAt={'2024-09-12 18:37:50'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>平均收款期（Average Collection Period, ACP）是指企业从销售商品或服务到收到客户付款所需的平均天数。该指标反映了企业应收账款的管理效率和客户支付的及时性。平均收款期越短，说明企业能够更快地回收现金，提高资金的周转效率；平均收款期越长，则可能表明企业在应收账款管理或客户信用控制方面存在问题。</p><p>主要特点包括：</p><ol><li><strong>反映收款效率</strong>：衡量企业应收账款的收款效率和客户支付的及时性。</li><li><strong>财务管理指标</strong>：用于评估企业资金周转情况和财务管理效果。</li><li><strong>信用控制</strong>：反映企业的信用政策和客户管理效果。</li><li><strong>与现金流相关</strong>：平均收款期越短，企业现金流越稳定；反之，现金流压力可能增大。</li></ol><p>平均收款期的计算公式：&nbsp;</p><p>平均收款期 = (应收账款总额/年销售收入)×365</p><p>或&nbsp;</p><p>平均收款期 = 应收账款总额/日销售收入)</p><p>应用示例： 假设某企业在年度结束时的应收账款总额为$100,000，年度销售收入为$1,000,000，则其平均收款期计算如下：&nbsp;</p><p>平均收款期 = (100,000/1,000,000)×365=36.5 天</p><p>这意味着该企业从销售到收款平均需要 36.5 天。</p>
