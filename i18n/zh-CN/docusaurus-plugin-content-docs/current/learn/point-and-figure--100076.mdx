# 点数图

<ArticleMeta id={100076} updatedAt={'2024-09-13 11:51:29'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>点数图 (P&amp;F 图) 是绘制股票、债券、商品或期货的价格走势图，而不考虑时间的流逝。</p><p>与其他类型的图表（如蜡烛图）不同，蜡烛图标记资产在确定的时间段内的运动程度，P&amp;F 图利用由堆叠的 X 或 O 组成的列，每个 X 或 O 代表一定金额的价格运动。X 表示价格上涨，O 表示价格下跌。</p><p>当查看 P&amp;F 图时，技术分析师仍然使用支撑和阻力等概念，以及其他模式。一些人认为，支撑和阻力水平以及突破在 P&amp;F 图上更清晰定义，因为它过滤了微小的价格变动，对虚假的突破更不敏感。</p>

<AIContent content={`<p><strong>定义：</strong>点数图（P&amp;F 图）是一种绘制股票、债券、商品或期货价格走势图的方法，不考虑时间的流逝。与其他类型的图表（如蜡烛图）不同，点数图利用由堆叠的 X 或 O 组成的列，每个 X 或 O 代表一定金额的价格运动。X 表示价格上涨，O 表示价格下跌。</p><p><strong>起源：</strong>点数图的起源可以追溯到 19 世纪末期，当时的交易员为了简化价格走势的分析，发明了这种图表。最早的点数图被称为 “点图”，后来逐渐演变成现在的点数图。</p><p><strong>类别与特点：</strong>点数图主要有两种类型：传统点数图和改良点数图。传统点数图使用固定的价格变动单位，而改良点数图则根据市场波动性调整价格变动单位。点数图的主要特点包括：1. 过滤微小的价格变动，减少噪音；2. 更清晰地显示支撑和阻力水平；3. 对虚假突破不敏感。</p><p><strong>具体案例：</strong>案例一：假设某股票的价格从 100 元上涨到 110 元，点数图会在价格每上涨 1 元时绘制一个 X，形成一列 X。案例二：如果该股票的价格从 110 元下跌到 105 元，点数图会在价格每下跌 1 元时绘制一个 O，形成一列 O。这些 X 和 O 的排列可以帮助投资者识别价格趋势和突破点。</p><p><strong>常见问题：</strong>1. 点数图是否适用于所有市场？点数图主要适用于波动较大的市场，如股票和期货市场。2. 如何确定价格变动单位？价格变动单位可以根据资产的波动性和投资者的偏好来确定。</p>`} id={100076} />
