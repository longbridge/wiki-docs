# 点数图

<ArticleMeta id={100076} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

点数图 (P&F 图) 是绘制股票、债券、商品或期货的价格走势图，而不考虑时间的流逝。与其他类型的图表（如蜡烛图）不同，蜡烛图标记资产在确定的时间段内的运动程度，P&F 图利用由堆叠的 X 或 O 组成的列，每个 X 或 O 代表一定金额的价格运动。X 表示价格上涨，O 表示价格下跌。当查看 P&F 图时，技术分析师仍然使用支撑和阻力等概念，以及其他模式。一些人认为，支撑和阻力水平以及突破在 P&F 图上更清晰定义，因为它过滤了微小的价格变动，对虚假的突破更不敏感。
