# 低/无证明贷款

<ArticleMeta id={102295} updatedAt={'2024-09-12 01:53:36'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>低/无证明贷款（Low/No Documentation Loan）是指借款人在申请贷款时只需提供很少或不提供收入、资产和就业证明的贷款类型。此类贷款通常用于自雇人士、自由职业者或收入波动较大的人群，他们可能难以提供传统贷款所需的全面财务文件。低/无证明贷款的审批标准较为宽松，借款人只需提供基本的个人信息和信用报告。这种贷款类型在 2008 年金融危机前非常流行，但由于其高风险性，许多贷款最终违约。现在，这类贷款的发放更加严格，贷款利率和费用也较高，以补偿贷款机构承担的更大风险。</p>
