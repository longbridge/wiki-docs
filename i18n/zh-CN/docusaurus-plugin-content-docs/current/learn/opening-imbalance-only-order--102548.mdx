# 开盘失衡仅限价单

<ArticleMeta id={102548} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

开盘失衡仅限价单（OIO）是纳斯达克开盘竞价过程中提供流动性的一种限价单。限价单是向经纪人下达的指令，以指定的股数和限价执行买入或卖出交易。
