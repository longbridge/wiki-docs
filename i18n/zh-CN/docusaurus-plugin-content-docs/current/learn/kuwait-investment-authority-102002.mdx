# 科威特投资局

<p><span style={{backgroundColor:"rgb(255,255,255)",color:"rgb(6,6,7)"}}>科威特投资局（KIA）是一个政府拥有的公司，成立于 1982 年，负责管理科威特的主权财富基金。KIA 起源于科威特投资委员会，并旨在管理政府的财政收入，这些收入主要来自于科威特石油储备的过剩收益。作为世界上第一个也是最古老的主权财富基金，KIA 的创建旨在减少科威特对石油收入的依赖，通过多元化投资策略来实现国家的长期财政稳定和增长。</span></p>
