# 平均蜡烛图

<ArticleMeta id={101562} updatedAt={'2023-09-26 13:58:58'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

平均蜡烛图使用平均价格数据创建一个日本蜡烛图，以过滤掉市场噪音。平均蜡烛图与标准蜡烛图具有一些相似之处，但是根据用于创建每根蜡烛的数值而不同。平均蜡烛图不像标准蜡烛图一样使用开盘价、最高价、最低价和收盘价，而是使用基于两期均值的修正公式。这使得图表呈现更平滑的外观，更容易发现趋势和反转，但也隐藏了缺口和一些价格数据。

<AIContent content={`<p><strong>定义：</strong>平均蜡烛图（Heikin-Ashi）是一种技术分析工具，通过使用平均价格数据来创建日本蜡烛图，以过滤掉市场噪音。与标准蜡烛图不同，平均蜡烛图使用基于两期均值的修正公式来计算每根蜡烛的开盘价、收盘价、最高价和最低价。</p><p><strong>起源：</strong>平均蜡烛图起源于日本，Heikin-Ashi 在日语中意为 “平均条”。这种图表类型在 20 世纪被引入西方金融市场，并逐渐被技术分析师所接受和使用。</p><p><strong>类别与特点：</strong>平均蜡烛图的主要特点是其平滑的外观，这使得趋势和反转更容易被识别。与标准蜡烛图相比，平均蜡烛图隐藏了价格缺口和一些细节数据，从而减少了市场噪音。其计算公式如下：<br/>1. 开盘价 = (前一根蜡烛的开盘价 + 前一根蜡烛的收盘价) / 2<br/>2. 收盘价 = (当前周期的开盘价 + 最高价 + 最低价 + 收盘价) / 4<br/>3. 最高价 = 当前周期的最高价、开盘价和收盘价中的最高值<br/>4. 最低价 = 当前周期的最低价、开盘价和收盘价中的最低值</p><p><strong>具体案例：</strong><br/>案例 1：在一个上升趋势中，平均蜡烛图通常会显示连续的绿色蜡烛，且蜡烛的下影线较短或没有下影线。这表明市场处于强势上升阶段。<br/>案例 2：在一个下降趋势中，平均蜡烛图通常会显示连续的红色蜡烛，且蜡烛的上影线较短或没有上影线。这表明市场处于强势下降阶段。</p><p><strong>常见问题：</strong><br/>1. <strong>平均蜡烛图是否适合短期交易？</strong> 平均蜡烛图更适合中长期趋势分析，因为它平滑了价格波动，可能会延迟信号。<br/>2. <strong>为什么平均蜡烛图看不到价格缺口？</strong> 由于其计算方法，平均蜡烛图隐藏了价格缺口，这有助于减少市场噪音，但也可能遗漏一些重要的价格信息。</p>`} id={101562} />
