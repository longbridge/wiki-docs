# 输入 - 输出分析

<ArticleMeta id={102403} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

输入 - 输出分析（I-O）是一种基于不同经济部门或产业之间相互依赖关系的宏观经济分析方法。这种方法常用于估计正面或负面经济冲击的影响，并分析对整个经济的涟漪效应。输入 - 输出经济分析最早由瓦西里·勒昂季耶夫（1906-1999）开发，他后来因在这一领域的工作而获得诺贝尔经济学奖。
