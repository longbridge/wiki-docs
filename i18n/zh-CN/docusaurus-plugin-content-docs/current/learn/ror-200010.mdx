# RoR

<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p><span style={{backgroundColor:"rgb(255,255,255)",color:"rgb(17,17,17)"}}>RoR 是 Rate of Return 的缩写，也即 “回报率”。</span></p><p><span style={{backgroundColor:"rgb(255,255,255)",color:"rgb(17,17,17)"}}>RoR 是投资在特定时期内的净收益或损失，以投资初始成本的百分比表示。</span></p>

<AIContent content={`<p><strong>定义：</strong><br/>RoR 是 Rate of Return 的缩写，也即 “回报率”。RoR 是投资在特定时期内的净收益或损失，以投资初始成本的百分比表示。它是衡量投资绩效的一个重要指标，帮助投资者评估其投资的盈利能力。</p><p><strong>起源：</strong><br/>回报率的概念可以追溯到早期的金融市场，当时投资者需要一种方法来衡量其投资的收益。随着金融市场的发展，回报率逐渐成为评估投资绩效的标准工具。</p><p><strong>类别与特点：</strong><br/>1. <strong>年化回报率（Annualized Rate of Return）：</strong>这是将投资的回报率标准化为每年的回报率，便于不同投资期限的比较。<br/>2. <strong>实际回报率（Real Rate of Return）：</strong>这是扣除通货膨胀影响后的回报率，更能反映投资的实际收益。<br/>3. <strong>名义回报率（Nominal Rate of Return）：</strong>这是未扣除通货膨胀影响的回报率，通常用于短期投资。</p><p><strong>具体案例：</strong><br/>1. <strong>股票投资：</strong>假设你在 2023 年初以 100 元购买了一只股票，到 2023 年底该股票价格上涨到 120 元，同时你还获得了 5 元的股息。那么，RoR = [(120 + 5 - 100) / 100] * 100% = 25%。<br/>2. <strong>房地产投资：</strong>假设你在 2020 年以 50 万元购买了一套房产，2024 年你以 70 万元卖出，同时期间你每年获得 2 万元的租金收入。那么，RoR = [(70 + (2*4) - 50) / 50] * 100% = 48%。</p><p><strong>常见问题：</strong><br/>1. <strong>如何计算回报率？</strong> 回报率的计算公式为：RoR = [(期末价值 - 初始投资 + 收益) / 初始投资] * 100%。<br/>2. <strong>回报率是否考虑风险？</strong> 回报率本身不考虑风险，投资者需要结合其他指标（如夏普比率）来评估风险调整后的回报。</p>`} id={200010} />
