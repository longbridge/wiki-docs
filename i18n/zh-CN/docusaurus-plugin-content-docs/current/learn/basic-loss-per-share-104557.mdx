# 基本每股亏损

<ArticleMeta id={104557} updatedAt={'2023-10-24 20:20:16'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

基本每股亏损是指企业每股普通股在一定时期内所产生的净亏损金额，是衡量公司经营业绩的重要指标之一。基本每股亏损越小，表明企业盈利能力越强。

<AIContent content={`<p><strong>定义：</strong>基本每股亏损（Basic Loss Per Share, 简称 LPS）是指企业每股普通股在一定时期内所产生的净亏损金额。它是衡量公司经营业绩的重要指标之一。基本每股亏损越小，表明企业盈利能力越强。</p><p><strong>起源：</strong>基本每股亏损的概念源自于会计和财务报表分析，最早在 20 世纪中期被广泛应用于企业财务报告中。随着资本市场的发展，投资者和分析师越来越重视这一指标，以评估公司的盈利能力和财务健康状况。</p><p><strong>类别与特点：</strong>基本每股亏损主要有以下几个特点：<ul><li>计算简单：基本每股亏损的计算公式为：<br/>基本每股亏损 = 净亏损 / 普通股加权平均股数</li><li>反映公司盈利能力：基本每股亏损越小，表明公司盈利能力越强，反之则表明公司可能面临财务困境。</li><li>时间敏感：该指标通常按季度或年度计算，能够及时反映公司在特定时期内的经营状况。</li></ul></p><p><strong>具体案例：</strong><ol><li>案例一：某公司在 2023 年第一季度的净亏损为 1000 万元，普通股加权平均股数为 500 万股，则该季度的基本每股亏损为：<br/>基本每股亏损 = 1000 万元 / 500 万股 = 2 元/股</li><li>案例二：另一家公司在 2023 年全年净亏损为 2000 万元，普通股加权平均股数为 1000 万股，则该年度的基本每股亏损为：<br/>基本每股亏损 = 2000 万元 / 1000 万股 = 2 元/股</li></ol></p><p><strong>常见问题：</strong><ul><li>为什么基本每股亏损是负数？<br/>因为它反映的是净亏损，亏损金额为负数，因此基本每股亏损也为负数。</li><li>基本每股亏损和稀释每股亏损有什么区别？<br/>基本每股亏损只考虑普通股，而稀释每股亏损还考虑了潜在的稀释性证券（如可转换债券、期权等）。</li></ul></p>`} id={104557} />
