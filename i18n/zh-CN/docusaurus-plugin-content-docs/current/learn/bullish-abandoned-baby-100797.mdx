# 牛市弃婴

<ArticleMeta id={100797} updatedAt={'2023-09-26 13:58:51'} alias={`[]`} />

牛市弃婴是一种蜡烛图形模式，用于向交易者发出下跌趋势反转的信号。它形成于下降趋势中，由三个价格条形组成。第一个是大幅下跌的蜡烛线，随后是一个跳空低开的十字蜡烛线。接下来的蜡烛线比十字蜡烛线开盘价更高，并迅速向上攀升。预期价格将会继续走高，因为该模式显示卖出力量至少暂时耗尽。牛市弃婴可与熊市弃婴模式相对比，后者标志着可能结束的上升趋势。
