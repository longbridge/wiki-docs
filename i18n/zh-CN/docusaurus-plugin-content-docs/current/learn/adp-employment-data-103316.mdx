# ADP 就业数据

<ArticleMeta id={103316} updatedAt={'2024-09-12 14:12:24'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

ADP 就业数据是由美国人力资源管理公司 Automatic Data Processing, Inc. (ADP) 发布的一项经济数据指标，用于衡量美国私营部门非农就业人数的变化。该数据通常在美国劳工部在每月初公布非农就业报告之前发布，因此被视为预测非农就业数据的一个重要指标。
