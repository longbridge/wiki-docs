# 委托代理问题

<ArticleMeta id={102143} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

委托代理问题是指一个人或团体与其代表有权代表其行事之间的优先权冲突。代理人可能会采取与委托人最佳利益相悖的行为。委托代理问题因委托人和代理人可能扮演的角色而异。它可以发生在任何一个资产所有权或委托人将该资产的直接控制权委托给另一方或代理人的情况。
