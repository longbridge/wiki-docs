# 或有价值权

<ArticleMeta id={100032} updatedAt={'2024-09-13 11:53:19'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>术语 “或有价值权（CVR）” 指的是一种权利，通常授予面临重组或收购的公司股东。这些权利确保股东在特定事件发生时获得一定的利益，通常在指定的时间框架内。</p><p>这些权利类似于期权，因为它们通常有一个到期日，超过此日期后，权利将不再适用。CVR 通常与公司股票的表现相关。</p>

<AIContent content={`<h2>或有价值权（CVR）</h2><p><strong>定义：</strong>术语 “或有价值权（Contingent Value Rights, CVR）” 指的是一种权利，通常授予面临重组或收购的公司股东。这些权利确保股东在特定事件发生时获得一定的利益，通常在指定的时间框架内。CVR 类似于期权，因为它们通常有一个到期日，超过此日期后，权利将不再适用。CVR 通常与公司股票的表现相关。</p><h3>起源：</h3><p>或有价值权的概念起源于 20 世纪 80 年代，当时公司重组和收购活动频繁。为了保护股东利益，企业开始引入 CVR 作为一种补偿机制，确保股东在特定条件下能够获得额外的收益。随着时间的推移，CVR 逐渐成为并购交易中的常见工具。</p><h3>类别与特点：</h3><p>CVR 可以分为两大类：现金支付型和股票支付型。<ul><li><strong>现金支付型：</strong>在特定事件发生时，股东将获得现金支付。这种类型的 CVR 通常与公司业绩指标或特定项目的成功挂钩。</li><li><strong>股票支付型：</strong>在特定事件发生时，股东将获得额外的公司股票。这种类型的 CVR 通常与公司股价表现相关。</li></ul>CVR 的主要特点包括：<ul><li><strong>条件性：</strong>CVR 的价值取决于特定事件的发生，如公司达到某一业绩目标或完成某一项目。</li><li><strong>时间限制：</strong>CVR 通常有一个到期日，超过此日期后，权利将不再适用。</li><li><strong>风险与回报：</strong>CVR 为股东提供了额外的潜在收益，但也伴随着一定的风险，因为特定事件可能不会发生。</li></ul></p><h3>具体案例：</h3><p><strong>案例一：</strong>在某次制药公司并购中，收购方承诺如果被收购公司的一款新药在未来两年内获得 FDA 批准，股东将获得每股 2 美元的现金支付。这种 CVR 确保了股东在新药成功上市时能够获得额外收益。<br/><strong>案例二：</strong>某科技公司在重组过程中，向股东发行了股票支付型 CVR。如果公司在未来三年内股价达到特定目标，股东将获得额外的公司股票。这种 CVR 激励股东支持公司的长期发展。</p><h3>常见问题：</h3><p><strong>1.
