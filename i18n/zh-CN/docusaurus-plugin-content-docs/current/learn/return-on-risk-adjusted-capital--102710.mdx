# 风险调整资本回报率

<p>风险调整资本回报率 (RORAC) 是一种常用的回报率衡量标准，用于金融分析中，根据风险资本评估各种项目、努力和投资。不同风险风险配置项目的 RORAC 值被计算后更容易进行比较。</p><p>RORAC 类似于净资产回报率 (ROE)，只不过分母根据项目的风险进行了调整。</p>
