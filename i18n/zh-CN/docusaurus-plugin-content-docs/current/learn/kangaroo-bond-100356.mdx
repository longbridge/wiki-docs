# 袋鼠债券

<p>袋鼠债券是指由非澳大利亚公司或机构在澳大利亚发行的以澳大利亚元计价的债券。这种债券通常是由外国公司、政府或国际组织在澳大利亚市场上发行的，目的是筹集澳大利亚元资金。袋鼠债券的发行者可以利用澳大利亚的资本市场和投资者基础，同时投资者也可以通过购买这种债券来分散投资组合并获得澳元资产。袋鼠债券的发行和交易受澳大利亚金融市场的监管。</p>
