# 权益成本

<ArticleMeta id={100855} updatedAt={'2023-09-25 22:58:51'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

权益成本是公司为确定一个投资是否符合资本回报要求所需的回报。公司通常将其作为投资回报的资本预算阈值。公司的权益成本代表了市场对拥有这一资产并承担所有权风险所要求的补偿。权益成本的传统公式为股利资本化模型和资本资产定价模型（CAPM）。

<AIContent content={`<p><strong>定义：</strong>权益成本是公司为确定一个投资是否符合资本回报要求所需的回报。公司通常将其作为投资回报的资本预算阈值。公司的权益成本代表了市场对拥有这一资产并承担所有权风险所要求的补偿。</p><p><strong>起源：</strong>权益成本的概念起源于 20 世纪中期，随着现代金融理论的发展而逐渐成型。特别是威廉·夏普在 1964 年提出的资本资产定价模型（CAPM），为计算权益成本提供了理论基础。</p><p><strong>类别与特点：</strong>权益成本主要通过两种方法计算：股利资本化模型和资本资产定价模型（CAPM）。<br/>1. <strong>股利资本化模型：</strong>该模型基于公司未来股利的现值来计算权益成本，公式为：<br/><code>权益成本 = (下一年预期股利 / 当前股价) + 股利增长率</code>。<br/>2. <strong>资本资产定价模型（CAPM）：</strong>该模型考虑了系统性风险，公式为：<br/><code>权益成本 = 无风险利率 + β * (市场回报率 - 无风险利率)</code>。</p><p><strong>具体案例：</strong><br/>1. <strong>案例一：</strong>假设某公司当前股价为 100 元，预计下一年股利为 5 元，股利增长率为 4%。根据股利资本化模型，权益成本为：<br/><code>权益成本 = (5 / 100) + 0.04 = 0.09 或 9%</code>。<br/>2. <strong>案例二：</strong>假设某公司β值为 1.2，无风险利率为 3%，市场回报率为 8%。根据 CAPM 模型，权益成本为：<br/><code>权益成本 = 0.03 + 1.2 * (0.08 - 0.03) = 0.09 或 9%</code>。</p><p><strong>常见问题：</strong><br/>1. <strong>如何选择合适的模型计算权益成本？</strong>选择模型取决于公司股利政策和市场数据的可获得性。如果公司定期支付股利且股利增长稳定，股利资本化模型较为适用；否则，CAPM 模型更为常用。<br/>2. <strong>权益成本是否固定不变？</strong>权益成本会随着市场条件、公司风险状况和无风险利率的变化而变化。</p>`} id={100855} />
