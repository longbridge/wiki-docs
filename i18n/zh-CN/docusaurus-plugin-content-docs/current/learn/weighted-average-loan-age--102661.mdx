# 加权平均贷款年龄 

<ArticleMeta id={102661} updatedAt={'2023-09-25 22:59:02'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>加权平均贷款年龄 (WALA) 是衡量按其余余额或贷款名义价值对每个抵押支持证券池中贷款年龄的平均数的指标，权重基于在每个到期日的每笔贷款的美元金额与池子的总金额的比例。</p>

<AIContent content={`<p><strong>定义：</strong>加权平均贷款年龄 (WALA) 是衡量按其余余额或贷款名义价值对每个抵押支持证券池中贷款年龄的平均数的指标，权重基于在每个到期日的每笔贷款的美元金额与池子的总金额的比例。</p><p><strong>起源：</strong>加权平均贷款年龄的概念起源于抵押支持证券（MBS）市场，旨在帮助投资者评估证券池中贷款的平均年龄。随着 MBS 市场的发展，WALA 成为衡量贷款池风险和预期收益的重要工具。</p><p><strong>类别与特点：</strong>WALA 主要用于评估抵押支持证券池中的贷款年龄。其特点包括：1. 反映贷款池的整体年龄结构；2. 帮助投资者评估贷款池的风险和预期收益；3. 通过加权计算，提供更准确的贷款年龄分布信息。WALA 的应用场景主要在 MBS 市场，投资者可以通过 WALA 了解贷款池的平均年龄，从而做出更明智的投资决策。</p><p><strong>具体案例：</strong>案例 1：某 MBS 池包含三笔贷款，金额分别为$100,000、$200,000 和$300,000，贷款年龄分别为 2 年、3 年和 5 年。WALA 计算如下：WALA = (100,000*2 + 200,000*3 + 300,000*5) / (100,000 + 200,000 + 300,000) = 4 年。案例 2：另一个 MBS 池包含四笔贷款，金额分别为$150,000、$250,000、$350,000 和$250,000，贷款年龄分别为 1 年、2 年、4 年和 6 年。WALA 计算如下：WALA = (150,000*1 + 250,000*2 + 350,000*4 + 250,000*6) / (150,000 + 250,000 + 350,000 + 250,000) = 3.75 年。</p><p><strong>常见问题：</strong>1. 为什么 WALA 对投资者重要？WALA 帮助投资者了解贷款池的平均年龄，从而评估其风险和预期收益。2. WALA 与加权平均贷款期限（WAM）有何不同？WALA 关注的是贷款的实际年龄，而 WAM 关注的是贷款的剩余期限。</p>`} id={102661} />
