# 平均收益率

<ArticleMeta id={101563} updatedAt={'2023-09-26 13:58:58'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

平均收益率是在指定的时间段内产生的一系列收益的简单数学平均值。平均收益率的计算方法与计算任何一组数字的简单平均值相同。将这些数字相加得到一个总数，然后将总数除以该组数字的数量。
