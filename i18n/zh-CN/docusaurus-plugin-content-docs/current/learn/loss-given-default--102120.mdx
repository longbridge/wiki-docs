# 损失给定违约

<ArticleMeta id={102120} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

损失给定违约（LGD）是指银行或其他金融机构在借款人违约时预计会损失的金额。LGD 以违约时的总敞口金额的百分比或潜在损失的单一美元价值表示。金融机构的总 LGD 是在对所有未偿贷款进行累积损失和敞口的审查后计算出的。
