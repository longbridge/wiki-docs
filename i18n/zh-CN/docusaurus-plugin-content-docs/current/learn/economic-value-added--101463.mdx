# 经济附加值

<ArticleMeta id={101463} updatedAt={'2024-09-13 10:41:21'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

经济附加值（EVA）是一种基于公司的净资产价值的财务绩效指标，计算方法是将税前经营利润减去其资本成本，再调整为现金基础上的税后金额。EVA 也可以被称为经济利润，因为它试图捕捉到公司的真正经济利润。这个指标是由管理咨询公司 Stern Value Management 构思出来的，最初的公司名称是 Stern Stewart & Co。

<AIContent content={`<h2>经济附加值（EVA）</h2><h3>定义</h3><p>经济附加值（EVA，Economic Value Added）是一种基于公司的净资产价值的财务绩效指标。它通过将税前经营利润减去资本成本，再调整为现金基础上的税后金额来计算。EVA 试图捕捉公司的真正经济利润，因此也被称为经济利润。</p><h3>起源</h3><p>EVA 由管理咨询公司 Stern Value Management（原名 Stern Stewart & Co）构思出来。该指标在 20 世纪 80 年代末和 90 年代初逐渐流行，成为衡量公司财务绩效的重要工具。</p><h3>类别与特点</h3><p>EVA 主要有以下几个特点：</p><ul><li><strong>真实反映经济利润：</strong>通过扣除资本成本，EVA 能够更真实地反映公司的经济利润，而不仅仅是会计利润。</li><li><strong>关注资本效率：</strong>通过考虑资本成本，EVA 鼓励公司更有效地利用资本，提高投资回报率。</li><li><strong>长期视角：</strong>EVA 强调长期价值创造，避免短期行为。</li></ul><h3>具体案例</h3><p><strong>案例一：</strong>某公司在某年度的税前经营利润为 500 万元，资本成本为 200 万元，税率为 30%。则该公司的 EVA 计算如下：</p><p>税后经营利润 = 500 万元 × (1 - 30%) = 350 万元</p><p>EVA = 350 万元 - 200 万元 = 150 万元</p><p>该公司在这一年度创造了 150 万元的经济附加值。</p><p><strong>案例二：</strong>另一家公司在某年度的税前经营利润为 1000 万元，资本成本为 800 万元，税率为 25%。则该公司的 EVA 计算如下：</p><p>税后经营利润 = 1000 万元 × (1 - 25%) = 750 万元</p><p>EVA = 750 万元 - 800 万元 = -50 万元</p><p>该公司在这一年度未能创造经济附加值，反而亏损了 50 万元。</p><h3>常见问题</h3><p><strong>问题一：</strong>为什么 EVA 比净利润更能反映公司的经济绩效？</p><p><strong>解答：</strong>因为 EVA 考虑了资本成本，而净利润没有考虑资本成本。EVA 能够更真实地反映公司为股东创造的经济价值。</p><p><strong>问题二：</strong>如何提高公司的 EVA？</p><p><strong>解答：</strong>公司可以通过提高经营利润、降低资本成本或更有效地利用资本来提高 EVA。</p>`} id={101463} />
