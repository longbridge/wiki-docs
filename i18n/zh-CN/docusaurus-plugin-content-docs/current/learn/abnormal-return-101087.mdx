# 异常收益

<ArticleMeta id={101087} updatedAt={'2023-09-25 22:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>异常收益是指投资组合或单个股票的实际收益超出或低于预期收益的部分，通常用于评估投资表现或市场反应。</p>

<AIContent content={`<p><strong>定义：</strong>异常收益是指投资组合或单个股票的实际收益超出或低于预期收益的部分。预期收益通常基于市场平均收益、风险调整后的收益或其他基准收益。异常收益常用于评估投资表现或市场反应。</p><p><strong>起源：</strong>异常收益的概念起源于金融学中的资本资产定价模型（CAPM）和有效市场假说（EMH）。CAPM 模型通过风险和预期收益的关系来预测资产的合理收益，而 EMH 则假设市场价格已经反映了所有可用信息，因此任何超出预期的收益都被视为异常。</p><p><strong>类别与特点：</strong>异常收益可以分为正异常收益和负异常收益。正异常收益表示实际收益高于预期收益，通常被视为投资成功的标志；负异常收益则表示实际收益低于预期收益，可能表明投资决策存在问题。异常收益的特点包括：1. 难以预测：由于市场信息的复杂性和不确定性，异常收益难以提前预测。2. 短期波动：异常收益通常是短期现象，长期来看市场会趋于均衡。3. 信息敏感：市场上的新信息（如公司财报、经济数据）会迅速反映在价格中，导致异常收益的产生。</p><p><strong>具体案例：</strong>案例一：某公司发布了超出市场预期的季度财报，导致其股票价格在短时间内大幅上涨，投资者因此获得了正异常收益。案例二：某国央行意外宣布加息，市场未能提前预料到这一信息，导致股市整体下跌，投资者因此遭受了负异常收益。</p><p><strong>常见问题：</strong>1. 如何计算异常收益？通常使用实际收益减去预期收益的方法。2. 异常收益是否可持续？由于市场趋于均衡，异常收益通常是短期现象，难以长期持续。</p>`} id={101087} />
