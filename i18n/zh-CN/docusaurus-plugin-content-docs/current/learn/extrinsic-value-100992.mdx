# 外在价值

<ArticleMeta id={100992} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

外在价值是指期权的市场价格 (即保费) 与其内在价值之间的差异。外在价值还是通过除了基础资产价格之外的因素来确定期权的价值。与外在价值相对的是内在价值，即期权的固有价值。
