# 联合申报测试

<ArticleMeta id={102025} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

联合申报测试是 IRS 的一项测试，潜在扶养对象必须通过该测试才能被另一个纳税人认为是扶养对象之一。联合申报测试规定，任何扶养对象不能与配偶一起申报联合纳税，同时仍然可以被他人认为是其它纳税人的扶养对象，如父母或监护人。然而，此规则有一个例外。由于申报扶养对象具有价值，IRS 制定了几项测试，如联合申报测试，以确保扶养对象不会被重复计数。
