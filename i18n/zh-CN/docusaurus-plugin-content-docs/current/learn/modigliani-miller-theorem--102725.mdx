# 莫迪利亚尼 - 米勒定理

<ArticleMeta id={102725} updatedAt={'2024-09-12 14:38:08'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>莫迪利亚尼 - 米勒定理（M&amp;M）指出，公司的市场价值应正确计算为其未来收益和基础资产的现值，并且与其资本结构无关。</p><p>该定理在基本层面上表明，在特定的假设条件下，一家公司是通过借债、发行股票或者重新投资利润来融资其增长是无关紧要的。</p><p>这个理论在 20 世纪 50 年代的发展对企业财务有着重大影响。</p>

<AIContent content={`<p><strong>定义：</strong>莫迪利亚尼 - 米勒定理（Modigliani-Miller Theorem，简称 M&amp;M 定理）指出，在一个完美市场中，公司的市场价值应正确计算为其未来收益和基础资产的现值，并且与其资本结构无关。换句话说，公司是通过借债、发行股票还是重新投资利润来融资其增长，对其市场价值没有影响。</p><p><strong>起源：</strong>莫迪利亚尼 - 米勒定理由弗兰科·莫迪利亚尼（Franco Modigliani）和梅顿·米勒（Merton Miller）在 1958 年提出。他们的研究在 20 世纪 50 年代对企业财务理论产生了重大影响，并在 1990 年获得了诺贝尔经济学奖。</p><p><strong>类别与特点：</strong>莫迪利亚尼 - 米勒定理主要分为两个命题：<br/>1. <strong>命题 I：</strong>在没有税收、破产成本和不对称信息的情况下，公司资本结构对其市场价值没有影响。<br/>2. <strong>命题 II：</strong>在考虑税收的情况下，负债融资可以带来税收盾，从而增加公司的市场价值。<br/>这些命题的特点是基于一系列严格的假设条件，如市场无摩擦、无税收、无破产成本等。</p><p><strong>具体案例：</strong><br/>1. <strong>案例一：</strong>假设公司 A 和公司 B 在同一行业，拥有相同的资产和未来收益预期。公司 A 完全通过股权融资，而公司 B 则通过 50% 的股权和 50% 的债务融资。根据 M&amp;M 定理命题 I，在没有税收和其他市场摩擦的情况下，两家公司的市场价值应相同。<br/>2. <strong>案例二：</strong>假设公司 C 在有税收的环境下，通过增加债务融资来利用税收盾。根据 M&amp;M 定理命题 II，公司 C 的市场价值将因税收盾的存在而增加。</p><p><strong>常见问题：</strong><br/>1. <strong>问：</strong>为什么在现实中公司资本结构会影响其市场价值？<br/><strong>答：</strong>因为现实市场中存在税收、破产成本、不对称信息等因素，这些都会影响公司的资本结构决策和市场价值。<br/>2. <strong>问：</strong>莫迪利亚尼 - 米勒定理的假设条件在现实中是否成立？<br/><strong>答：</strong>大多数情况下，这些假设条件在现实中并不完全成立，但该定理为理解资本结构的影响提供了一个理论基础。</p>`} id={102725} />
