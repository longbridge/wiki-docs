# 股东增值

<ArticleMeta id={100496} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

股东增值是指公司创造的超额经营利润与资本成本或资本成本之间的差额。基本计算公式为税后净经营利润 (NOPAT) 减去资本成本，资本成本根据公司的加权平均资本成本来计算。
