# M2

<ArticleMeta id={100049} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

M2 是指某一个时间内，货币资产的总量，又被称为广义货币供应量。它是金融系统对货币供应总量的估计，一般包括所有人持有的现金以及存款账户、储蓄账户和其他短期储蓄工具（如定期存单）。 
在货币供应量中，还包括了 M0 和 M1。
M0 = 流动中的现金；
M1 = M0 + 商业银行活期存款，又被称为狭义货币。
而 M2 = M1 + 商业银行定期存款。
