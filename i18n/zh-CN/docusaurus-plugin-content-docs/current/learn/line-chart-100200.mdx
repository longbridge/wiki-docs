# 折线图

<p>折折线图通过将一系列数据点用线连接起来，以图形的方式表示资产价格随时间的变化。这是金融中最基本的图表类型，通常只显示一个证券随时间的收盘价格。折线图可以用于任何时间框架，但通常使用每日价格变动。</p>
