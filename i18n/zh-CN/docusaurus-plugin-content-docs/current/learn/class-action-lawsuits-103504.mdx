# 集体诉讼

<ArticleMeta id={103504} updatedAt={'2023-10-07 02:24:55'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

集体诉讼是指一组人联合起来对某个机构或公司提起诉讼，以维护他们的共同利益。集体诉讼通常是由某个代表人或代表律师领导的，他们代表所有受影响的人进行谈判或诉讼。
