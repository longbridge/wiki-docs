# A 类普通股

<ArticleMeta id={103209} updatedAt={'2024-09-12 14:13:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

A 类普通股是一种股票类别，通常具有更高的投票权和更高的优先权。A 类股票通常不向公众发行，而是由公司的创始人、高管和董事会成员持有。这种股票通常与 B 类普通股等其他类别的股票一起发行。
