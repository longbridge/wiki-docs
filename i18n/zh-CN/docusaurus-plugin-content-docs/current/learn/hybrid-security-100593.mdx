# 混合证券

<ArticleMeta id={100593} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

混合证券是指一种将两种或更多不同金融工具组合而成的单一金融工具。混合证券通常被称为 “混合品”，一般结合债务和股权特征。最常见的混合证券类型是可转换债券，它具有普通债券的特点，但其价格受到可转换成的股票价格的影响。
