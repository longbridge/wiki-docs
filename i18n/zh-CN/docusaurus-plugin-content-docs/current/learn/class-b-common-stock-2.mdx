# B 类普通股

<ArticleMeta id={2} updatedAt={'2024-09-12 21:11:43'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>B 类普通股是指具有普通股权益的股票，但在投票权和分红权等方面相对较低的股票。B 类普通股通常是由一般投资者所持有的股票，其权益和优先权相对较低。但在某些公司中，B 类普通股可能拥有更高的投票权，具体取决于公司的章程和治理结构。</p>
