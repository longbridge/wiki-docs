# 直布罗陀镑

<ArticleMeta id={101746} updatedAt={'2024-09-12 19:06:57'} alias={`[]`} />

直布罗陀镑（GIP）是直布罗陀国家的官方货币。直布罗陀镑与英镑的汇率为 1 比 1，采取固定汇率制。
