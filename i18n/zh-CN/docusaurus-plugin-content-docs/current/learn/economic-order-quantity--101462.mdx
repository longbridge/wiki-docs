# 经济订购量

<ArticleMeta id={101462} updatedAt={'2024-09-12 19:41:20'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

经济订购量（EOQ）是指公司应该购买的理想单位数量，以满足需求，同时尽量减少库存成本，如持有成本、短缺成本和订购成本。这个生产调度模型是由福特·W·哈里斯在 1913 年开发的，并经过了不断的改进。经济订购量公式假设需求、订购和持有成本都保持不变。

<AIContent content={`<p><strong>定义：</strong>经济订购量（Economic Order Quantity，简称 EOQ）是指公司在满足需求的同时，尽量减少库存成本（包括持有成本、短缺成本和订购成本）所应购买的理想单位数量。EOQ 模型帮助企业确定最佳的订购量，以实现库存管理的最优化。</p><p><strong>起源：</strong>经济订购量模型最早由福特·W·哈里斯于 1913 年提出。尽管最初的模型较为简单，但随着时间的推移，研究人员和实践者对其进行了多次改进，使其更加适应现代企业的需求。</p><p><strong>类别与特点：</strong>EOQ 模型主要有以下几个特点：<ul><li><strong>固定需求：</strong>假设需求量是已知且恒定的。</li><li><strong>固定订购成本：</strong>每次订购的成本是固定的，不随订购数量变化。</li><li><strong>固定持有成本：</strong>持有库存的成本是固定的，通常与库存量成正比。</li></ul>此外，EOQ 模型还可以根据不同的需求和成本结构进行调整，如考虑批量折扣、变动需求等。</p><p><strong>具体案例：</strong><ol><li><strong>案例一：</strong>某零售公司每年需要销售 12000 个产品，每次订购的固定成本为 100 元，持有每个产品的年成本为 2 元。通过 EOQ 公式计算，该公司每次应订购的数量为：<br/>EOQ = √(2DS/H) = √(2*12000*100/2) = √1200000 = 1095 个。<br/>这样可以最小化总库存成本。</li><li><strong>案例二：</strong>一家制造企业每年需要 5000 个零部件，每次订购的固定成本为 50 元，持有每个零部件的年成本为 1 元。通过 EOQ 公式计算，该企业每次应订购的数量为：<br/>EOQ = √(2DS/H) = √(2*5000*50/1) = √500000 = 707 个。<br/>这样可以有效减少库存持有和订购成本。</li></ol></p><p><strong>常见问题：</strong><ul><li><strong>问题一：</strong>如果需求量不恒定，EOQ 模型是否仍然适用？<br/><strong>解答：</strong>EOQ 模型假设需求量是恒定的。如果需求量波动较大，可能需要使用其他库存管理模型，如动态订购模型。</li><li><strong>问题二：</strong>如何处理批量折扣？<br/><strong>解答：</strong>在存在批量折扣的情况下，可以调整 EOQ 公式，考虑不同订购量下的总成本，选择最小化总成本的订购量。</li></ul></p>`} id={101462} />
