# 无杠杆贝塔

<ArticleMeta id={101369} updatedAt={'2023-09-26 13:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>无杠杆贝塔是衡量公司在没有任何债务情况下的市场风险的一种指标。它反映了公司自身业务的风险，而不考虑其资本结构的影响。无杠杆贝塔通过剔除公司财务杠杆的影响，使得投资者可以更准确地评估公司的基本风险。</p>

<AIContent content={`<p><strong>定义：</strong>无杠杆贝塔（Unlevered Beta）是衡量公司在没有任何债务情况下的市场风险的一种指标。它反映了公司自身业务的风险，而不考虑其资本结构的影响。无杠杆贝塔通过剔除公司财务杠杆的影响，使得投资者可以更准确地评估公司的基本风险。</p><p><strong>起源：</strong>无杠杆贝塔的概念源自资本资产定价模型（CAPM），该模型由威廉·夏普（William Sharpe）在 20 世纪 60 年代提出。CAPM 用于评估资产的预期收益率，其中贝塔系数是衡量资产相对于市场波动的风险指标。无杠杆贝塔则进一步剔除了财务杠杆的影响，使得投资者可以更纯粹地评估公司的业务风险。</p><p><strong>类别与特点：</strong>无杠杆贝塔主要有以下特点：<ul><li>剔除财务杠杆：无杠杆贝塔不考虑公司的债务，专注于公司的业务风险。</li><li>行业对比：无杠杆贝塔可以用于不同公司的行业对比，因为它排除了资本结构的影响。</li><li>风险评估：无杠杆贝塔帮助投资者更准确地评估公司的基本风险。</li></ul></p><p><strong>具体案例：</strong><ol><li>假设公司 A 的杠杆贝塔为 1.5，债务权益比为 0.5，企业税率为 30%。无杠杆贝塔的计算公式为：<br/>无杠杆贝塔 = 杠杆贝塔 / [1 + (1 - 税率) * 债务权益比]<br/>代入数据：无杠杆贝塔 = 1.5 / [1 + (1 - 0.3) * 0.5] = 1.2</li><li>公司 B 和公司 C 在同一行业，分别有不同的资本结构。通过计算无杠杆贝塔，投资者可以更准确地比较两家公司在没有债务情况下的业务风险。</li></ol></p><p><strong>常见问题：</strong><ul><li><strong>问：</strong>无杠杆贝塔和杠杆贝塔有什么区别？<br/><strong>答：</strong>无杠杆贝塔剔除了公司的债务影响，专注于业务风险；杠杆贝塔则包括了公司的财务杠杆。</li><li><strong>问：</strong>为什么要使用无杠杆贝塔？<br/><strong>答：</strong>使用无杠杆贝塔可以更准确地评估公司的基本业务风险，特别是在进行行业对比时。</li></ul></p>`} id={101369} />
