# 订单审核跟踪系统

<ArticleMeta id={102508} updatedAt={'2023-09-25 22:59:02'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

订单审核跟踪系统（OATS）是由金融业监管机构金融业监管局（FINRA）建立的自动化计算机系统。它用于记录与全国市场系统（NMS）上交易的所有权益，包括场外交易（OTC）股票的订单、报价和其他相关交易数据。该系统简化了订单从初始接收到最终执行或取消的过程，以便于追踪或审计目的。

<AIContent content={`<h2>订单审核跟踪系统（OATS）</h2><h3>定义</h3><p>订单审核跟踪系统（Order Audit Trail System，简称 OATS）是由金融业监管局（FINRA）建立的自动化计算机系统。该系统用于记录与全国市场系统（NMS）上交易的所有权益，包括场外交易（OTC）股票的订单、报价和其他相关交易数据。OATS 简化了订单从初始接收到最终执行或取消的过程，以便于追踪或审计目的。</p><h3>起源</h3><p>OATS 于 1998 年由 FINRA 推出，旨在提高市场透明度和交易活动的可追溯性。其初衷是为了应对当时市场上存在的交易不透明和监管难题，通过自动化系统记录所有交易活动，确保市场的公平和有效运行。</p><h3>类别与特点</h3><p>OATS 主要分为两类：一类是针对 NMS 股票的订单记录，另一类是针对 OTC 股票的订单记录。其特点包括：</p><ul><li>自动化：系统自动记录所有订单和交易数据，减少人为错误。</li><li>全面性：覆盖所有 NMS 和 OTC 股票的交易活动。</li><li>可追溯性：每个订单从接收到执行或取消的全过程都被详细记录，便于审计和监管。</li></ul><h3>具体案例</h3><p>案例一：某投资者通过经纪商 A 下达了一笔购买某 NMS 股票的订单。OATS 系统会自动记录该订单的接收时间、订单类型、数量、价格等详细信息，并在订单执行或取消后更新记录。这使得监管机构可以随时追踪该订单的整个生命周期。</p><p>案例二：某经纪商 B 在场外市场（OTC）进行了一笔股票交易。OATS 系统同样会记录该交易的所有相关数据，包括报价、订单接收时间、执行时间等。这确保了即使在场外市场，交易活动也能被有效追踪和审计。</p><h3>常见问题</h3><p>问题一：OATS 系统是否会影响订单的执行速度？<br/>回答：OATS 系统的设计初衷是为了记录和追踪订单数据，不会直接影响订单的执行速度。</p><p>问题二：所有经纪商都需要使用 OATS 系统吗？<br/>回答：是的，所有在 NMS 和 OTC 市场上进行交易的经纪商都需要遵守 OATS 系统的规定，确保所有订单和交易数据被准确记录。</p>`} id={102508} />
