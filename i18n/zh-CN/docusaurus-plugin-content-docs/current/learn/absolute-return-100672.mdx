# 绝对回报

<ArticleMeta id={100672} updatedAt={'2023-09-26 13:58:51'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

绝对回报是指在指定期间内资产实现的回报。这个指标以百分比表示了资产（如股票或共同基金）在给定期间内的升值或贬值。绝对回报与相对回报的不同之处在于，它关注的是特定资产的回报，并不将其与其他指标或基准进行比较。
