# Probability Density Function 

<ArticleMeta id={102316} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

The probability density function (PDF) is a statistical expression that defines the probability that some outcome will occur. In this function, the probability is the percentage of a dataset's distribution that falls between two criteria.PDF is used by financial analysts to understand how returns are distributed in order to evaluate the risk and expectations of investment prices and returns.
