If the order is not executed prior to the end of the week, it will be automatically canceled. <span style={{backgroundColor:"rgb(255,255,255)",color:"rgb(17,17,17)"}}>More common order types include market orders, limit orders, and Good 'Til Canceled (GTC) orders.</span></p>
