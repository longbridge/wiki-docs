# Total Return Swap

A total return swap is aÂ swap agreement in which one party makes payments based on a set rate, either fixed or variable, while the other party makes payments based on the return of an underlying asset, which includes both the income it generates and any capital gains. In total return swaps, the underlying asset, referred to as the reference asset, is usually an equity index, a basket of loans, or bonds. The asset is owned by the party receiving the set rate payment.
