# Value Added Monthly Index 

<ArticleMeta id={101159} updatedAt={'2023-09-25 22:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>Value Added Monthly Index (VAMI) is a metric used to measure the performance of an investment portfolio or fund. It tracks the value changes of an initial investment on a monthly basis to show the cumulative returns. VAMI typically starts with a fixed initial amount (e.g., $1000) and adjusts this amount according to the monthly return rates.&nbsp;</p>
