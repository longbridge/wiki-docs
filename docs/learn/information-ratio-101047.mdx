# Information Ratio

<ArticleMeta id={101047} updatedAt={'2023-09-25 22:58:54'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>The Information Ratio is a metric used to evaluate the performance of an investment portfolio. It reflects the relationship between the portfolio's excess return and the nonsystematic risk it takes on. Specifically, it is the ratio of the portfolio's excess return (i.e., returns above the benchmark) to the tracking error (i.e., the standard deviation of the difference between the portfolio returns and the benchmark returns). A higher Information Ratio indicates that the portfolio achieves more excess return for the same level of risk.</p>
