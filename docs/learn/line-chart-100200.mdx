<p><span style={{backgroundColor:"rgb(255,255,255)",color:"rgb(17,17,17)"}}>A line chart graphically represents an asset's price over time by connecting a series of data points with a line.</span> This is the most basic type of chart used in finance, and it typically only depicts a security's closing prices over time.
