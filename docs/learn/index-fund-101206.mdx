# Index Fund

<ArticleMeta id={101206} updatedAt={'2024-09-13 10:54:14'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>An index fund is a passively managed investment fund that aims to replicate the performance of a specific stock market index (such as the S&amp;P 500 Index, Nasdaq 100 Index, Hang Seng Index, etc.). The fund achieves returns similar to the index by holding the same stocks that make up the index.</p>
