# One-Touch Option

<ArticleMeta id={101698} updatedAt={'2023-09-25 22:58:58'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

A one-touch option pays a premium to the holder of the option if the spot rate reaches the strike price at any time prior to option expiration.
