# Program Evaluation Review Technique 

<ArticleMeta id={102684} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

A program evaluation review technique (PERT) chart is a graphical representation of a project's timeline that displays all of the individual tasks necessary to complete the project. As a project management tool, the PERT chart is often preferred to the Gantt chart because it identifies task dependencies. However, a PERT chart can be more difficult to interpret.
