# Bermuda Option

<ArticleMeta id={101289} updatedAt={'2023-09-26 13:58:54'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

A Bermuda option is a type of exotic options contract that can only be exercised on predetermined dates—often on one day each month.A spin on American-style options, which permit holders to exercise early at any time, Bermudian options allow investors to buy or sell a security or underlying asset at a preset price on a set of specific dates as well as the option's expiration date.
