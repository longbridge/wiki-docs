# DuPont Analysis

<ArticleMeta id={100392} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

The DuPont analysis is a framework for analyzing fundamental performance popularized by the DuPont Corporation. DuPont analysis is a useful technique used to decompose the different drivers of return on equity (ROE).
