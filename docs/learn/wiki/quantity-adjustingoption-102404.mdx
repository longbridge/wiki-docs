# Quantity-Adjusting Option 

<ArticleMeta id={102404} updatedAt={'2023-09-26 13:59:00'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

A quantity-adjusting option, commonly called a Quanto option, is a cross-currency derivative in which the underlying asset is denominated in one currency but settlement is made in another currency at a predetermined exchange rate.Another name for these options is a guaranteed exchange rate option. Quanto options come in both call and put varieties.
