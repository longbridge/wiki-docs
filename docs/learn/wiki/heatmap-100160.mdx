# Heatmap

<ArticleMeta id={100160} updatedAt={'2023-09-26 13:58:47'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

A heatmap is a two-dimensional visual representation of data using colors, where the colors all represent different values.A heatmap can be used with all sorts of data, from the real estate market representing the number of foreclosures to the spreads of credit default swaps (CDS) to webpage analysis reflecting the number of hits a website receives.
