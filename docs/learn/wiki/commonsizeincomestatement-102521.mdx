# Common Size Income Statement

<ArticleMeta id={102521} updatedAt={'2023-09-26 13:59:02'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

A common size income statement is an income statement in which each line item is expressed as a percentage of the value of revenue or sales. It is used for vertical analysis, in which each line item in a financial statement is represented as a percentage of a base figure within the statement.Common size financial statements help to analyze and compare a company's performance over several periods with varying sales figures. The common size percentages can be subsequently compared to those of competitors to determine how the company is performing relative to the industry.
