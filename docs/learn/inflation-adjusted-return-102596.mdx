# Inflation-Adjusted Return

<ArticleMeta id={102596} updatedAt={'2023-09-26 13:59:02'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>The inflation-adjusted return is the measure of return that takes into account the time period's inflation rate. The purpose of the inflation-adjusted return metric is to reveal the return on an investment after removing the effects of inflation.</p><p>Removing the effects of inflation from the return of an investment allows the investor to see the true earning potential of the security without external economic forces. The inflation-adjusted return is also known as the real rate of return or required rate of return adjusted for inflation.</p>
