# Installment Debt

<ArticleMeta id={100428} updatedAt={'2023-09-26 13:58:47'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>Installment debt refers to a type of debt where the borrower repays the borrowed amount in scheduled installments over a set period. Each installment typically includes both principal and interest. Installment debt is commonly used in personal loans, auto loans, mortgages, and student loans.</p>
