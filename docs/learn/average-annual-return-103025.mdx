# average annual return

<ArticleMeta id={103025} updatedAt={'2023-09-29 02:20:38'} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

The average annualized return refers to the average annualized return of an asset or investment portfolio over a certain period of time. It is an important investment indicator used to evaluate the long-term performance of assets or investment portfolios. The average annualized return can be calculated based on different time periods, usually on an annual basis.
