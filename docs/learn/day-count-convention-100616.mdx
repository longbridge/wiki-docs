# Day-Count Convention

<p><span style={{backgroundColor:"rgba(0,0,0,0.05)",color:"rgb(36,41,47)"}}>The Day-Count Convention is a standardized method used in financial markets to calculate interest. It defines how days should be counted when computing interest, with different conventions applicable to various financial instruments and markets. Common day-count conventions include Actual/Actual, Actual/360, Actual/365, 30/360, and 30/365, among others. For instance, the Actual/Actual convention calculates interest based on the actual number of days and the actual number of days in the year, while the 30/360 convention assumes each month has 30 days and each year has 360 days. The choice of day-count convention affects the interest calculation for bonds, loans, and other financial instruments, making it crucial to specify the convention in financial contracts.</span></p>
