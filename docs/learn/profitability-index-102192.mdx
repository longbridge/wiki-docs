# Profitability Index

<p>The Profitability Index (PI) is a financial metric used to evaluate the feasibility of an investment project. It represents the ratio of the present value of future cash flows generated by the project to the initial investment cost. PI is used to determine whether a project is worth investing in, typically in capital budgeting decisions. When PI is greater than 1, it indicates that the project's expected returns exceed the initial investment cost, making the project feasible; when PI is less than 1, it indicates that the project's expected returns are less than the initial investment cost, making the project unfeasible.</p><p>The formula for calculating the Profitability Index is:<br/>Profitability&nbsp;Index&nbsp;(PI) = Present&nbsp;Value&nbsp;of&nbsp;Projectâ€™s&nbsp;Cash&nbsp;Flows/Initial&nbsp;Investment&nbsp;Cost</p><p>Key characteristics include:</p><p>Ratio Analysis: Assesses the profitability of investment projects in ratio form, facilitating comparison of different projects.<br/>Decision Basis: Assists companies in selecting the optimal investment projects in capital budgeting.<br/>Time Value Consideration: Takes the time value of money into account, unlike some other evaluation methods.<br/>Simple and Intuitive: Simple to calculate and easy to understand, making it practical for application.<br/>Example of Profitability Index application:<br/>Suppose an investment project has a present value of cash flows amounting to $2 million and an initial investment cost of $1.5 million. The project's PI would be:<br/>PI = 2&nbsp;million&nbsp;USD/1.5&nbsp;million&nbsp;USD = 1.33<br/>Since PI is greater than 1, the project is considered feasible and profitable.</p>
