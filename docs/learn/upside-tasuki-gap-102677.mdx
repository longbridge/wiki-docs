<p>An Upside Tasuki Gap is a three-bar candlestick formation that is commonly used to signal the continuation of the current trend.</p><ol><li>The first bar is a large white/green candlestick within a defined uptrend.</li><li>The second bar is another white/green candlestick with an opening price that has gapped above the close of the previous bar.</li><li>The third bar is a black/red candlestick that partially closes the gap between the first two bars.</li></ol>
