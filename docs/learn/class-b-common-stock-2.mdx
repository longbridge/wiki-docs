# Class B Common Stock

<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>Class B common stock refers to stocks that have common stock equity, but relatively lower voting rights and dividend rights. Class B common stock is usually held by ordinary investors, with lower rights and privileges.</p>
