# Binomial Distribution

<ArticleMeta id={100409} updatedAt={'2024-09-13 11:35:16'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>The Binomial Distribution is a discrete probability distribution that describes the number of successes in a fixed number of independent trials, where each trial has only two possible outcomes: "success" and "failure." The binomial distribution is defined by two parameters: the number of trials n and the probability of success p in each trial.</p>
