# Time-Weighted Rate of Return 

<ArticleMeta id={102589} updatedAt={'2023-09-26 13:59:02'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

<p>The time-weighted rate of return (TWR) is a measure of the compound rate of growth in a portfolio. The TWR measure is often used to compare the returns of investment managers because it eliminates the distorting effects on growth rates created by inflows and outflows of money. The time-weighted return breaks up the return on an investment portfolio into separate intervals based on whether money was added or withdrawn from the fund.</p><p>The time-weighted return measure is also called the geometric mean return, which is a complicated way of stating that the returns for each sub-period are multiplied by each other.</p>
