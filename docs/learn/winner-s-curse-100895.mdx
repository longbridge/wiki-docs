# Winner'S Curse

<ArticleMeta id={100895} updatedAt={'2023-09-26 13:58:51'} alias={`[]`} />
<div className='border-solid border-b border-t-0 my-4 border-[var(--ifm-color-gray-300)]' />

The winner's curse is a tendency for the winning bid in an auction to exceed the intrinsic value or true worth of an item. The gap in auctioned versus intrinsic value can typically be attributed to incomplete information, emotions, or a variety of other subjective factors that may influenceÂ bidders.In general, subjective factors usually create a value gap because the bidder faces a difficult time determining and rationalizing an item's true intrinsic value. As a result, the largest overestimation of an item's value ends up winning the auction.
